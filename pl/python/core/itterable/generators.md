# генераторы

- перебирать можно только один раз
- функции max, min, list, sum, set могут вызываться с генераторами (перебирать можно только один раз)
- смысл в том что бы создавать последовательности без потери памяти
- len() нельзя использовать
- создать list можно только с помощью list
- нельзя обращаться по индексу

```python
# это не картеж
a = (x ** 2 for x in range(6))

next(a) # 1
next(a) # 4
next(a) # 9
next(a) # 16
next(a) # 25
```

# yield

вместо return

```python
def get_list():
  for x in [1,2,3,4]:
    yield x

a = get_list()

for x in a:
  print(x)

```
