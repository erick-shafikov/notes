Вывод

```python
print(1, 5, 7)
# sep
a = 1
b = 2
c = 25
# sep разделитель нужно хотя бы 2 аргумента
print(a, b, c, sep=" | ")
# end переводит курсор на новую строку по умолчанию
print(a, b, c, end=" | ")
# в консоли отобразятся в строку
print('hello', end=' ')
print('world')
# форматированный вывод
print("Координаты точки: x = ", x , "; y=", y , sep='')
# F-строка
print(f"Координаты точки: x = {x}; y = {y}")
```
