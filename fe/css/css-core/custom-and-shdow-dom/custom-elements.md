# @scope

Позволяет определить стили для поддеревьев

```scss
 @scope (scope root) to (scope limit) {
  rulesets
}
```

Может использоваться вместе с псевдо классом scope

## :state()

для определенного состояния компонента

```scss
.labeled-checkbox {
  border: dashed red;
}
.labeled-checkbox:state(checked) {
  border: solid;
}
```
