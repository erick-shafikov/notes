# проваливание атрибутов

Атрибуты имеют свойство проваливаться, то есть если у компонента определить стандартные атрибуты они применятся к template компоненты

!!!сработает если у template один дочерний элемент

```vue
<script setup>
// позволят предотвратить проваливание атрибутов
defineOptions({
  inheritAttrs: false,
});
</script>

<template>
  <button class="button">
    <!-- ... -->
    <!-- доступ для атрибутов, которые проваливаются -->
    {{ $attrs }}
  </button>
</template>
```
