# Broadcasting в NumPy

Broadcasting — механизм, позволяющий выполнять арифметику над массивами **разных форм** без физического копирования данных.

## Правила (применяются справа налево по осям)

1. Сравниваем `shape` справа налево.
2. Ось совместима, если:
   - размеры равны: `N == N`
   - один из размеров равен 1: `N` и `1`
   - ось отсутствует у одного из массивов (считается `1`)
3. Размерность `1` **растягивается** до размера партнёра.
4. Размерности `2`, `3`, `5` и т.д. растягивать **нельзя**.

## Совместимые размерности

| A    | B    | Результат | Почему                      |
|------|------|-----------|-----------------------------|
| `N`  | `N`  | `N`       | равны                        |
| `N`  | `1`  | `N`       | `1` растягивается до `N`    |
| `N`  | —    | `N`       | отсутствие ≡ `1`            |

## Несовместимые размерности

| A     | B    | Ошибка | Почему                    |
|-------|------|--------|---------------------------|
| `100` | `2`  | ✗      | ни одна не равна `1`      |
| `100` | `50` | ✗      | ни одна не равна `1`      |
| `4`   | `3`  | ✗      | ни одна не равна `1`      |

```python
import numpy as np

a = np.ones((3, 4))
b = np.ones((3, 2))
a + b  # ValueError: несовместимые оси 4 и 2
```

## Примеры

### Скаляр + массив (самый простой случай)

```python
a = np.array([1, 2, 3])   # shape (3,)
b = 10                     # shape () — скаляр

a + b  # [11, 12, 13]
# b "растягивается" до (3,): [10, 10, 10]
```

### Вектор-строка + вектор-столбец → матрица

```python
row = np.array([[1, 2, 3]])   # shape (1, 3)
col = np.array([[10],         # shape (3, 1)
                [20],
                [30]])

row + col
# shape выходного: (3, 3)
# [[11, 12, 13],
#  [21, 22, 23],
#  [31, 32, 33]]
```

Как работает выравнивание осей:
```
row:  (1, 3)   →  растягивается по оси 0 до (3, 3)
col:  (3, 1)   →  растягивается по оси 1 до (3, 3)
```

### 2D-матрица + 1D-вектор

```python
A = np.array([[1, 2, 3],   # shape (2, 3)
              [4, 5, 6]])

v = np.array([10, 20, 30]) # shape (3,)

A + v
# [[11, 22, 33],
#  [14, 25, 36]]
```

Выравнивание справа налево:
```
A:  (2, 3)
v:     (3,)  →  считается (1, 3)  →  растягивается до (2, 3)
```

### Нормализация строк матрицы

```python
A = np.array([[1, 2, 3],
              [4, 5, 6]])           # shape (2, 3)

row_means = A.mean(axis=1)          # shape (2,)  — ОШИБКА, если прибавить напрямую
row_means = A.mean(axis=1, keepdims=True)  # shape (2, 1)

A - row_means
# [[-1.,  0.,  1.],
#  [-1.,  0.,  1.]]
# shape (2,1) растягивается до (2,3)
```

> `keepdims=True` — частый приём: сохраняет ось, чтобы broadcasting сработал корректно.

### 3D-тензор + 1D-вектор (батч)

```python
batch = np.zeros((32, 224, 224))  # 32 изображения 224×224, shape (32, 224, 224)
bias  = np.array([0.1])           # shape (1,)

batch + bias  # shape (32, 224, 224) — bias добавляется к каждому пикселю
```

Выравнивание:
```
batch:  (32, 224, 224)
bias:          (  1,)  →  (1, 1, 1)  →  (32, 224, 224)
```

## Диагностика ошибок

```python
a = np.ones((4, 3))
b = np.ones((4,))

a + b  # ValueError!
```

Почему ошибка:
```
a:  (4, 3)
b:     (4,)   ←  выравнивание справа: ось 1 = 4 vs 3 — несовместимо!
```

Исправление:
```python
b = b.reshape(4, 1)  # или b[:, np.newaxis]
a + b  # shape (4, 3) — ок
```

---

## Визуальная схема алгоритма

```
A.shape = (5, 1, 4)
B.shape =    (3, 1)

Выравниваем справа:
        A:  5  1  4
        B:     3  1
            ↓  ↓  ↓
совместимо? 5  3  4  →  да (1 растягивается)
результат:  (5, 3, 4)
```
