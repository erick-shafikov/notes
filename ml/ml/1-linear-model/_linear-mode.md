# Линейная модель

Если есть некоторая линейная функция $a(x) = kx + b$, то её можно представить в виде суммы базисных функций: $a(x) = θ_1 f_1(x) + θ_2 f_2(x)$, где $f_1(x) = x$, $f_2(x) = 1$, что в общем виде даёт $a(x) = \sum_{i=0}^{n} θ_i f_i(x)$.

Как пример можно взять аппроксимацию функции $y = \sin(x) + 0.3x + ε(x)$. Предположим, что можно построить модель с одним параметром $x$, тогда расширенное линейно независимое признаковое пространство будет: $f_0(x) = 1$, $f_1(x) = x$, $f_2(x) = x^2$, ..., $f_n(x) = x^n$, и функцию можно представить как $a(x) = θ_0 + θ_1 x + θ_2 x^2 + ... + θ_n x^n$, где $θ = (θ_0, θ_1, ..., θ_n)^T$, либо в виде $a(x) = \langle θ, f(x) \rangle$.

Минус полинома высокой степени в том, что на больших данных возникает эффект переобучения (overfitting).

Если рассматривать квадратичную функцию потерь $L(a, y) = (a(x_i) - y_i)^2$, тогда показатель качества принимает вид $Q(w) = \frac{1}{n} \sum_{i=1}^{n} (a(x_i) - y_i)^2 = \frac{1}{n} \|Xw - Y\|^2$, где $X$ — матрица признаков, $w$ — вектор весов, $Y$ — вектор ответов. Минимизируя потери получаем градиент $∇Q(w) = \frac{2}{n} X^T (Xw - Y)$, приравнивая его к нулю $X^T (Xw - Y) = 0$, откуда $w = (X^T X)^{-1} X^T Y$.

# Гиперплоскость в задачах бинарной классификации

Представим $a(x) = kx + b$ в виде $w_1 x_1 + w_2 x_2 - w_0 = 0$, тогда $w = (w_1, w_2)^T$ и $x = (x_1, x_2)^T$, и функцию можно записать как $\langle w, x \rangle - w_0 = 0$, где $\langle w, x \rangle = w^T x$. Вектор $w$ ортогонален разделяющей гиперплоскости, а $w_0$ — смещение.

Решающая функция имеет вид $a(x) = \text{sign}(\langle w, x \rangle - w_0)$. Введём понятие отступа $M_i = y_i \langle w, x_i \rangle$: если $M_i > 0$ — классификация верная, если $M_i < 0$ — ошибка. Тогда функция потерь может быть записана как $L(a, y) = [y \cdot a(x) < 0]$, а показатель качества $Q(w) = \sum_{i=1}^{n} [M_i < 0]$.

Поиск решения можно выполнить с помощью алгоритма Розенблатта:

- инициализируем веса $w = 0$,
- далее повторяем несколько эпох, проходя по выборке $(x_i, y_i)$,
  - вычисляя отступ $M_i = y_i \langle w, x_i \rangle$, и если $M_i \leq 0$,
    - обновляем веса по правилу $w = w + η y_i x_i$, где $η$ — шаг обучения;
  - процесс продолжается до отсутствия ошибок или достижения лимита итераций. Показатель качества при этом $Q(w) = \sum_{i=1}^{n} [M_i \leq 0]$.

Так как функцию потерь с индикатором дифференцировать нельзя, используют её гладкие приближения:

- $V(M) = \max(0, 1 - M)$ (SVM),
- $L(M) = \log(1 + e^{-M})$ (логистическая),
- $Q(M) = (1 - M)^2$ (квадратичная),
- $E(M) = e^{-M}$ (экспоненциальная).

На примере квадратичной функции: $L_i(w) = (1 - M_i)^2 = (1 - y_i \langle w, x_i \rangle)^2$, тогда $Q(w) = \sum_{i=1}^{n} (1 - y_i \langle w, x_i \rangle)^2$, и решение можно записать как $w^T = \left( \sum_{i=1}^{n} x_i^T y_i \right) \left( \sum_{i=1}^{n} x_i x_i^T \right)^{-1}$.
