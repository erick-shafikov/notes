# Стохастический градиентный спуск (Stochastic Average Gradient SAG)

- Вход:
- - X - выборка, η - шаг обучения
- Выход:
- - ω - вектор весовых коэффициентов
- Инициализация весов ω некоторыми начальными значениями
- Инициализация хранилища градиентов для каждого объекта выборки $G_i = 0$
- Вычисление функционала качества $Q(ω) = 1 / l * \sum_{i=1}^{l}L_i(ω)$
- цикл:
- - Случайный выбор $x_k$ из $X^l$
- - Вычисление функции потерь $ε_k = L_k(ω)$
- - вычисление градиента $g_k = ∇L_k(ω, x_k)$
- - Обновление сохраненного градиента для объекта: $G_k = g_k$
- - Обновление среднего градиента $G = 1 / l * \sum_{i=1}^{l} G_i$
- - Шаг градиентного алгоритма $ω = ω - η * G$
- - Пересчет функционала качества $Q = λ * ε_k + (1 - λ) * Q$
- - Пока Q и/или не достигнут заданных значений

Плюсы:

- более плавный спуск по сравнению со SGD
- меньшая дисперсия обновлений градиента
- использует информацию обо всей выборке через накопленные градиенты

Минусы:

- большой объем памяти (нужно хранить градиенты для всей выборки)
- более дорогие обновления по сравнению со SGD
- сложнее реализация

Отличие от SGD:

- SGD использует градиент одного случайного объекта и имеет высокую шумность обновлений
- SAG использует усреднение накопленных градиентов по всей выборке, поэтому обновления более стабильны и менее шумные
