# Наивный Байесовский классификатор, логистическая регрессия, Метод максимального правдоподобия

При бинарной классификации вместо жесткого ответа используется вероятность принадлежности к классу. Пусть $P(y | x, ω)$ — вероятность принадлежности объекта $x$ классу $y$.

Задача — найти такие параметры $ω$, которые максимизируют правдоподобие выборки:
$ω^* = argmax ∏_{i = 1}^l P(y_i | x_i, ω)$

Предполагается независимость объектов, тогда:
$P(y_1, y_2, ... y_l | x_1, x_2, ... x_l, ω) = ∏_{i = 1}^l P(y_i | x_i, ω)$

Для удобства оптимизации переходят к логарифму:
$log L(ω) = Σ_{i=1}^l log(P(y_i | x_i, ω)) → max$

Эквивалентно можно минимизировать функцию потерь:
$Q(a, X^l) = - Σ_{i=1}^l log(P(y_i | x_i, ω)) → min$

Таким образом, вероятностная задача максимизации правдоподобия сводится к задаче минимизации функции потерь.

Рассмотрим логистическую регрессию. Функция потерь:
$L(M) = log(1 + e ^ {-M})$, где $M_i = y_i * <ω, x_i>$

Тогда:
$Σ_{i=1}^l log(1 + e ^ {-M_i}) = - Σ_{i=1}^l log(P(y_i | x_i, ω)) → min$

Вероятность класса задается сигмоидой:
$P(y = 1 | x, ω) = 1 / (1 + e^{-<ω, x>})$

Для многоклассового случая используется функция softmax:
$P(y = k | x, ω) = exp(<ω_k, x>) / Σ_{j} exp(<ω_j, x>)$

Наивный Байесовский классификатор предполагает условную независимость признаков и использует вероятностную модель для классификации

- Наивный Байесовский классификатор для задач регрессии

```python
import numpy as np

f = lambda x: 0.5 * x + 0.2 * x ** 2 - 0.05 * x ** 3 + 0.2 * np.sin(4 * x) - 2.5
p_prod = lambda X, Y, w, D: - 1 / D * (Y - X @ w) @ X  # нахождение точки максимума  ПРВ

coord_x = np.arange(-4.0, 6.0, 0.1)
x_train = np.array([[_x ** i for i in range(4)] for _x in coord_x])
y_train = f(coord_x)

N = 10000
w = np.array([.0, .0, .0, .0])
k = np.array([.1, .001, .0001, .00001])

D = np.var(coord_x)  # Дисперсия

Q = np.mean((x_train @ w - y_train) ** 2)

for _ in range(N):
    grad = p_prod(x_train, y_train, w, D)
    w -= k * grad

Q = np.mean((x_train @ w - y_train) ** 2)
```
