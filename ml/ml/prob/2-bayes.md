# Байесовский вывод (Наивный Байес)

Задача **линейной классификации**: $y \in \mathcal{Y} = \{-1, +1\}$, обучающая выборка $X^l = (x_i, y_i)_{i=1}^l \sim P(x, y)$.

Задача: восстановить $P(y \mid x) = ?$ — вероятность класса по объекту.

По теореме Байеса:

$$P(y \mid x) = \frac{P(y) \cdot P(x \mid y)}{P(x)}$$

где:

- $P(y)$ — априорная вероятность класса $y$
- $P(x \mid y)$ — распределение признаков $x$ для объектов класса $y \in \{-1, +1\}$
- $P(x)$ — маргинальное распределение объектов, не зависит от $y$

Поскольку $P(x)$ не зависит от $y$, задача классификации сводится к:

$$a(x) = \arg\max_{y \in \mathcal{Y}} P(y \mid x) = \arg\max_{y \in \mathcal{Y}} P(y) \cdot P(x \mid y)$$

Это **оптимальный байесовский классификатор**.

С учётом штрафов $\lambda$ за ошибки классификации:

$$a(x) = \arg\max_{y \in \mathcal{Y}} \lambda_y \cdot P(y) \cdot P(x \mid y)$$

**Байесовский риск** — ожидаемые потери от ошибок классификатора:

$$Q(a) = \sum_y \lambda_y \int \mathbf{1}[a(x) \neq y] \cdot p(x, y)\, dx$$

На практике $P(x, y)$ неизвестно, поэтому оцениваем $\hat{P}(y)$ и $\hat{p}(x \mid y)$ по выборке.

## Наивное байесовское допущение

**Наивный Байес** — упрощение: предполагается, что признаки $\xi_1, \xi_2, \ldots, \xi_n$ **условно независимы** при данном $y$:

$$x = [\xi_1,\, \xi_2,\, \ldots,\, \xi_n]^T, \quad p(x \mid y) = \prod_{i=1}^n p(\xi_i \mid y)$$

Каждое $p(\xi_i \mid y)$ — одномерное распределение, что существенно упрощает оценку.

Классификатор принимает вид:

$$a(x) = \arg\max_{y \in \mathcal{Y}} \lambda_y \cdot P(y) \cdot \prod_{i=1}^n p(\xi_i \mid y)$$

Переходим к логарифму (произведение → сумма):

$$a(x) = \arg\max_{y \in \mathcal{Y}} \left(\ln \lambda_y \cdot P(y) + \sum_{i=1}^n \ln p(\xi_i \mid y)\right)$$

## Пример Gaussian Naive Bayes

Выборка объектов с двумя непрерывными признаками: ширина $\ell$ и длина $w$:

| №   | $\ell$ (ширина) | $w$ (длина) | $y$ |
| --- | --------------- | ----------- | --- |
| 1   | 10              | 50          | +1  |
| 2   | 20              | 30          | −1  |
| …   | …               | …           | …   |
| 10  | 7               | 35          | −1  |

Дискриминантная функция для каждого класса:

$$a(\ell, w \mid y = -1) = \ln\!\left(\lambda_{-1} \cdot \hat{P}(y{=}-1)\right) + \ln\frac{1}{2\pi\,\hat{\sigma}_\ell^-\hat{\sigma}_w^-} - \frac{(\ell - \hat{m}_\ell^-)^2}{2{\hat{\sigma}_\ell^-}^2} - \frac{(w - \hat{m}_w^-)^2}{2{\hat{\sigma}_w^-}^2}$$

$$a(\ell, w \mid y = +1) = \ln\!\left(\lambda_{+1} \cdot \hat{P}(y{=}+1)\right) + \ln\frac{1}{2\pi\,\hat{\sigma}_\ell^+\hat{\sigma}_w^+} - \frac{(\ell - \hat{m}_\ell^+)^2}{2{\hat{\sigma}_\ell^+}^2} - \frac{(w - \hat{m}_w^+)^2}{2{\hat{\sigma}_w^+}^2}$$

Решение: объект $x = (\ell, w)$ относится к классу $y^* = \arg\max_{y} a(\ell, w \mid y)$.

Если каждый признак распределён нормально при данном классе:

$$p(\ell \mid y) = \frac{1}{\sqrt{2\pi}\,\sigma_\ell^y} \exp\!\left\{-\frac{(\ell - m_\ell^y)^2}{2{\sigma_\ell^y}^2}\right\}, \qquad p(w \mid y) = \frac{1}{\sqrt{2\pi}\,\sigma_w^y} \exp\!\left\{-\frac{(w - m_w^y)^2}{2{\sigma_w^y}^2}\right\}$$

Параметры оцениваются по выборке отдельно для каждого класса.

**Оценки математических ожиданий:**

$$\hat{m}_\ell^- = \frac{\ell_1^- + \ell_2^- + \cdots + \ell_{n^-}^-}{n^-}, \qquad \hat{m}_\ell^+ = \frac{\ell_1^+ + \ell_2^+ + \cdots + \ell_{n^+}^+}{n^+}$$

$$\hat{m}_w^- = \frac{w_1^- + w_2^- + \cdots + w_{n^-}^-}{n^-}, \qquad \hat{m}_w^+ = \frac{w_1^+ + w_2^+ + \cdots + w_{n^+}^+}{n^+}$$

**Оценки дисперсий** (несмещённая):

$$\hat{\sigma}^2 = \frac{1}{N-1}\sum_{i=1}^N (x_i - \hat{m})^2$$

Для каждого признака и класса отдельно:

$${\hat{\sigma}_\ell^-}^2 = \frac{(\ell_1^- - \hat{m}_\ell^-)^2 + \cdots + (\ell_{n^-}^- - \hat{m}_\ell^-)^2}{n^-}, \qquad {\hat{\sigma}_\ell^+}^2 = \frac{\cdots}{n^+}$$

Аналогично ${\hat{\sigma}_w^-}^2$ и ${\hat{\sigma}_w^+}^2$.

**Совместное распределение** признаков по классу (в силу независимости):

$$\hat{p}(\ell, w \mid y) = \hat{p}(\ell \mid y) \cdot \hat{p}(w \mid y) = \frac{1}{2\pi\,\sigma_\ell^y\,\sigma_w^y} \exp\!\left\{-\frac{(\ell - m_\ell^y)^2}{2{\sigma_\ell^y}^2} - \frac{(w - m_w^y)^2}{2{\sigma_w^y}^2}\right\}$$

**Априорные вероятности** оцениваются эмпирически:

$$\hat{P}(y = -1) = \frac{n^-}{n^- + n^+}, \qquad \hat{P}(y = +1) = \frac{n^+}{n^- + n^+}$$

- Наивный Байесовский классификатор для задач регрессии

```python
import numpy as np

f = lambda x: 0.5 * x + 0.2 * x ** 2 - 0.05 * x ** 3 + 0.2 * np.sin(4 * x) - 2.5
p_prod = lambda X, Y, w, D: - 1 / D * (Y - X @ w) @ X  # нахождение точки максимума  ПРВ

coord_x = np.arange(-4.0, 6.0, 0.1)
x_train = np.array([[_x ** i for i in range(4)] for _x in coord_x])
y_train = f(coord_x)

N = 10000
w = np.array([.0, .0, .0, .0])
k = np.array([.1, .001, .0001, .00001])

D = np.var(coord_x)  # Дисперсия

Q = np.mean((x_train @ w - y_train) ** 2)

for _ in range(N):
    grad = p_prod(x_train, y_train, w, D)
    w -= k * grad

Q = np.mean((x_train @ w - y_train) ** 2)
```
