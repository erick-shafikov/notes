# Байесовский вывод (Наивный Байес)

Задача **линейной классификации**: $y \in \mathcal{Y} = \{-1, +1\}$, обучающая выборка $X^l = (x_i, y_i)_{i=1}^l \sim P(x, y)$.

Задача: восстановить $P(y \mid x) = ?$ — вероятность класса по объекту.

По теореме Байеса:

$$P(y \mid x) = \frac{P(y) \cdot P(x \mid y)}{P(x)}$$

где:

- $P(y)$ — априорная вероятность класса $y$
- $P(x \mid y)$ — распределение признаков $x$ для объектов класса $y \in \{-1, +1\}$
- $P(x)$ — маргинальное распределение объектов, не зависит от $y$

Поскольку $P(x)$ не зависит от $y$, задача классификации сводится к:

$$a(x) = \arg\max_{y \in \mathcal{Y}} P(y \mid x) = \arg\max_{y \in \mathcal{Y}} P(y) \cdot P(x \mid y)$$

Это **оптимальный байесовский классификатор**.

С учётом штрафов $\lambda$ за ошибки классификации:

$$a(x) = \arg\max_{y \in \mathcal{Y}} \lambda_y \cdot P(y) \cdot P(x \mid y)$$

$\lambda_y$ — цена ошибки при предсказании класса $y$: насколько дорого обходится пропустить объект этого класса. Например, в медицинской диагностике ошибочно предсказать «здоров» при реальной болезни ($y = +1$) гораздо опаснее, чем наоборот, поэтому $\lambda_{+1} \gg \lambda_{-1}$. Больший штраф «тянет» классификатор в сторону этого класса — даже при небольшой вероятности $P(y) \cdot P(x \mid y)$ объект будет отнесён к нему, лишь бы не допустить дорогой ошибки.

**Байесовский риск** — ожидаемые потери от ошибок классификатора:

$$Q(a) = \sum_y \lambda_y \int \mathbf{1}[a(x) \neq y] \cdot p(x, y)\, dx$$

На практике $P(x, y)$ неизвестно, поэтому оцениваем $\hat{P}(y)$ и $\hat{p}(x \mid y)$ по выборке.

# пример с выборкой единичной размерностью

Пусть есть обучающая выборка — один признак $x$ и два класса:

| $x$ | $y$ |
| --- | --- |
| 1   | −1  |
| 2   | −1  |
| 3   | −1  |
| 6   | +1  |
| 7   | +1  |
| 8   | +1  |

**Априорные вероятности (частоты).** Всего объектов $N = 6$, по три на каждый класс:

$$P(y = -1) = \frac{3}{6} = 0.5, \qquad P(y = +1) = \frac{3}{6} = 0.5$$

Здесь используются обычные частоты.

**Оцениваем распределения признака.**

Для класса $-1$ данные: $1, 2, 3$:

$$m_- = \frac{1+2+3}{3} = 2, \qquad \sigma_-^2 = \frac{(1-2)^2 + (2-2)^2 + (3-2)^2}{3} = \frac{2}{3}$$

Для класса $+1$ данные: $6, 7, 8$:

$$m_+ = 7, \qquad \sigma_+^2 = \frac{(6-7)^2 + (7-7)^2 + (8-7)^2}{3} = \frac{2}{3}$$

**Параметры модели:**

$$\omega = (m_-,\, \sigma_-,\, m_+,\, \sigma_+,\, P(y=-1),\, P(y=+1)) = \left(2,\, \sqrt{\tfrac{2}{3}},\, 7,\, \sqrt{\tfrac{2}{3}},\, 0.5,\, 0.5\right)$$

**Новый объект** $x = 2.5$.

Берём новый объект $x = 2.5$ и подставляем его в плотность распределения с уже обученными параметрами $\omega$. В Gaussian Naive Bayes параметры:

$$\omega = \{m_y, \sigma_y, P(y)\}$$

Для нашего примера:

$$\omega_- = (m_- = 2,\; \sigma_- = \sqrt{2/3}), \qquad \omega_+ = (m_+ = 7,\; \sigma_+ = \sqrt{2/3})$$

Gaussian NB предполагает:

$$p(x \mid y) = \frac{1}{\sigma_y\sqrt{2\pi}} \exp\!\left(-\frac{(x - m_y)^2}{2\sigma_y^2}\right)$$

Мы вычисляем $p(x = 2.5 \mid y = -1,\, \omega_-)$ — то есть насколько вероятно увидеть значение $2.5$, если объект принадлежит классу $-1$. Подставляем $x = 2.5$, $m_- = 2$, $\sigma_-^2 = \tfrac{2}{3}$:

$$p(2.5 \mid y=-1) = \frac{1}{\sqrt{2\pi} \cdot \sqrt{2/3}} \exp\!\left(-\frac{(2.5-2)^2}{2 \cdot 2/3}\right) \approx 0.41$$

Результат $0.41$ — это **не вероятность класса**, а плотность (likelihood): насколько типичен объект для данного класса.

Для класса $-1$: среднее близко ($m_- = 2$), поэтому $p(x = 2.5 \mid y = -1)$ большое.

Для класса $+1$ подставляем $x = 2.5$, $m_+ = 7$, $\sigma_+^2 = \tfrac{2}{3}$:

$$p(2.5 \mid y=+1) = \frac{1}{\sqrt{2\pi} \cdot \sqrt{2/3}} \exp\!\left(-\frac{(2.5-7)^2}{2 \cdot 2/3}\right) = \frac{1}{\sqrt{2\pi} \cdot \sqrt{2/3}} \exp\!\left(-\frac{20.25}{4/3}\right) \approx 1.24 \times 10^{-7}$$

Среднее далеко ($m_+ = 7$), поэтому правдоподобие почти нулевое.

**Применяем [теорему Байеса](../../../math/prob-math-stat/probability/basics/5-bayes-formula.md)** — переходим от likelihood к вероятности класса:

$$P(y \mid x) = \frac{P(y)\, p(x \mid y)}{\sum_{y'} P(y')\, p(x \mid y')}$$

Подставив вычисленные значения $p(x \mid -1) = 0.41$, $p(x \mid +1) = 1.24 \times 10^{-7}$:

$$P(y = -1 \mid x) = \frac{0.5 \cdot 0.41}{0.5 \cdot 0.41 + 0.5 \cdot 0.00001} \approx 0.99997$$

Важное различие: $p(x \mid y)$ отвечает на вопрос «насколько типичен объект для класса», а $P(y \mid x)$ — «какова вероятность класса».

**Главное понимание.** Частоты используются только для оценки параметров модели ($m$, $\sigma$, $P(y)$), а затем предполагается непрерывное распределение:

$$x \mid y \sim \mathcal{N}(m_y,\, \sigma_y^2)$$

Именно эта модель позволяет считать вероятность для любых новых $x$, даже тех, которых не было в данных.

Без модели, если считать только частотами:

$$P(y \mid x) = \frac{\#(x, y)}{\#(x)}$$

для $x = 2.5$ в данных нет такого объекта, и получится $\tfrac{0}{0}$. Модель распределения решает эту проблему.

**Откуда берётся произведение.** В примере выше был один признак. Если признаков много:

$$x = (x_1, x_2, \dots, x_n)$$

то в общем случае $p(x \mid y) = p(x_1, x_2, \dots, x_n \mid y)$ — это сложная многомерная плотность с ковариациями и зависимостями между признаками (матрица $\Sigma_y$).

В Naive Bayes делается ключевое упрощение — предположение о **условной независимости** признаков:

$$x_i \perp x_j \mid y$$

Тогда многомерная плотность распадается в произведение одномерных:

$$p(x \mid y) = \prod_{j=1}^n p(x_j \mid y)$$

Вместо одной сложной многомерной плотности обучается $n$ простых одномерных распределений на каждый класс. В гауссовском случае каждый признак моделируется отдельно:

$$p(x_j \mid y) = \mathcal{N}(m_j^y,\, (\sigma_j^y)^2)$$

и тогда:

$$p(x \mid y) = \prod_j \mathcal{N}(x_j \mid m_j^y,\, \sigma_j^y)$$

После логарифмирования произведение превращается в сумму:

$$\ln p(x \mid y) = \sum_j \ln p(x_j \mid y)$$

Это то, что появляется в дискриминантной функции. Интуиция: каждый признак «голосует» за класс независимо — один может тянуть к $+1$, другой к $-1$, итог складывается как сумма вкладов.

Метод хорошо работает, если признаки почти независимы или их зависимость не критична. Плохо работает при сильной корреляции признаков (например, рост и вес).

# задача для нескольких выборок

**Наивный Байес** — упрощение: предполагается, что признаки $\xi_1, \xi_2, \ldots, \xi_n$ **условно независимы** при данном $y$:

$$x = [\xi_1,\, \xi_2,\, \ldots,\, \xi_n]^T, \quad p(x \mid y) = \prod_{i=1}^n p(\xi_i \mid y)$$

Каждое $p(\xi_i \mid y)$ — одномерное распределение, что существенно упрощает оценку.

Классификатор принимает вид:

$$a(x) = \arg\max_{y \in \mathcal{Y}} \lambda_y \cdot P(y) \cdot \prod_{i=1}^n p(\xi_i \mid y)$$

Переходим к логарифму (произведение → сумма):

$$a(x) = \arg\max_{y \in \mathcal{Y}} \left(\ln \lambda_y \cdot P(y) + \sum_{i=1}^n \ln p(\xi_i \mid y)\right)$$

Множитель $\lambda_y \cdot P(y)$ остаётся одним слагаемым, потому что он не зависит от индекса признака $i$ — это скаляр, фиксированный для каждого класса $y$. Под произведение по $i$ (и, соответственно, под сумму после логарифмирования) попадают только $p(\xi_i \mid y)$, каждое из которых зависит от конкретного признака.

Выборка объектов с двумя непрерывными признаками: ширина $\ell$ и длина $w$:

| №   | $\ell$ (ширина) | $w$ (длина) | $y$ |
| --- | --------------- | ----------- | --- |
| 1   | 10              | 50          | +1  |
| 2   | 20              | 30          | −1  |
| …   | …               | …           | …   |
| 10  | 7               | 35          | −1  |

Подставляем произведение одномерных гауссиан и раскрываем логарифм:

$$\ln p(\ell, w \mid y) = \ln p(\ell \mid y) + \ln p(w \mid y)$$

$$= \ln \frac{1}{\sqrt{2\pi}\,\sigma_\ell^y} - \frac{(\ell - m_\ell^y)^2}{2{\sigma_\ell^y}^2} + \ln \frac{1}{\sqrt{2\pi}\,\sigma_w^y} - \frac{(w - m_w^y)^2}{2{\sigma_w^y}^2}$$

$$= \ln \frac{1}{2\pi\,\sigma_\ell^y\,\sigma_w^y} - \frac{(\ell - m_\ell^y)^2}{2{\sigma_\ell^y}^2} - \frac{(w - m_w^y)^2}{2{\sigma_w^y}^2}$$

Дискриминантная функция для каждого класса:

$$a(\ell, w \mid y = -1) = \ln\!\left(\lambda_{-1} \cdot \hat{P}(y{=}-1)\right) + \ln\frac{1}{2\pi\,\hat{\sigma}_\ell^-\hat{\sigma}_w^-} - \frac{(\ell - \hat{m}_\ell^-)^2}{2{\hat{\sigma}_\ell^-}^2} - \frac{(w - \hat{m}_w^-)^2}{2{\hat{\sigma}_w^-}^2}$$

$$a(\ell, w \mid y = +1) = \ln\!\left(\lambda_{+1} \cdot \hat{P}(y{=}+1)\right) + \ln\frac{1}{2\pi\,\hat{\sigma}_\ell^+\hat{\sigma}_w^+} - \frac{(\ell - \hat{m}_\ell^+)^2}{2{\hat{\sigma}_\ell^+}^2} - \frac{(w - \hat{m}_w^+)^2}{2{\hat{\sigma}_w^+}^2}$$

Решение: объект $x = (\ell, w)$ относится к классу $y^* = \arg\max_{y} a(\ell, w \mid y)$.

Если каждый признак распределён нормально при данном классе:

$$p(\ell \mid y) = \frac{1}{\sqrt{2\pi}\,\sigma_\ell^y} \exp\!\left\{-\frac{(\ell - m_\ell^y)^2}{2{\sigma_\ell^y}^2}\right\}, \qquad p(w \mid y) = \frac{1}{\sqrt{2\pi}\,\sigma_w^y} \exp\!\left\{-\frac{(w - m_w^y)^2}{2{\sigma_w^y}^2}\right\}$$

Параметры оцениваются по выборке отдельно для каждого класса.

**Оценки математических ожиданий:**

$$\hat{m}_\ell^- = \frac{\ell_1^- + \ell_2^- + \cdots + \ell_{n^-}^-}{n^-}, \qquad \hat{m}_\ell^+ = \frac{\ell_1^+ + \ell_2^+ + \cdots + \ell_{n^+}^+}{n^+}$$

$$\hat{m}_w^- = \frac{w_1^- + w_2^- + \cdots + w_{n^-}^-}{n^-}, \qquad \hat{m}_w^+ = \frac{w_1^+ + w_2^+ + \cdots + w_{n^+}^+}{n^+}$$

**Оценки дисперсий** (несмещённая):

$$\hat{\sigma}^2 = \frac{1}{N-1}\sum_{i=1}^N (x_i - \hat{m})^2$$

Для каждого признака и класса отдельно:

$${\hat{\sigma}_\ell^-}^2 = \frac{(\ell_1^- - \hat{m}_\ell^-)^2 + \cdots + (\ell_{n^-}^- - \hat{m}_\ell^-)^2}{n^-}, \qquad {\hat{\sigma}_\ell^+}^2 = \frac{\cdots}{n^+}$$

Аналогично ${\hat{\sigma}_w^-}^2$ и ${\hat{\sigma}_w^+}^2$.

**Совместное распределение** признаков по классу (в силу независимости):

$$\hat{p}(\ell, w \mid y) = \hat{p}(\ell \mid y) \cdot \hat{p}(w \mid y) = \frac{1}{2\pi\,\sigma_\ell^y\,\sigma_w^y} \exp\!\left\{-\frac{(\ell - m_\ell^y)^2}{2{\sigma_\ell^y}^2} - \frac{(w - m_w^y)^2}{2{\sigma_w^y}^2}\right\}$$

**Априорные вероятности** оцениваются эмпирически:

$$\hat{P}(y = -1) = \frac{n^-}{n^- + n^+}, \qquad \hat{P}(y = +1) = \frac{n^+}{n^- + n^+}$$

**Числовой пример.** Возьмём малую выборку из 6 объектов:

| №   | $\ell$ | $w$ | $y$ |
| --- | ------ | --- | --- |
| 1   | 10     | 50  | +1  |
| 2   | 20     | 30  | −1  |
| 3   | 12     | 45  | +1  |
| 4   | 18     | 35  | −1  |
| 5   | 8      | 55  | +1  |
| 6   | 22     | 25  | −1  |

Класс $-1$: объекты 2, 4, 6; класс $+1$: объекты 1, 3, 5.

**Шаг 1. Оценки математических ожиданий:**

$$\hat{m}_\ell^- = \frac{20+18+22}{3} = 20, \qquad \hat{m}_w^- = \frac{30+35+25}{3} = 30$$

$$\hat{m}_\ell^+ = \frac{10+12+8}{3} = 10, \qquad \hat{m}_w^+ = \frac{50+45+55}{3} = 50$$

**Шаг 2. Оценки дисперсий:**

$${\hat{\sigma}_\ell^-}^2 = \frac{(20-20)^2+(18-20)^2+(22-20)^2}{3} = \frac{0+4+4}{3} = \tfrac{8}{3}, \qquad {\hat{\sigma}_w^-}^2 = \frac{0+25+25}{3} = \tfrac{50}{3}$$

$${\hat{\sigma}_\ell^+}^2 = \frac{(10-10)^2+(12-10)^2+(8-10)^2}{3} = \tfrac{8}{3}, \qquad {\hat{\sigma}_w^+}^2 = \tfrac{50}{3}$$

**Шаг 3. Априорные вероятности:**

$$\hat{P}(y=-1) = \hat{P}(y=+1) = \tfrac{3}{6} = 0.5$$

**Шаг 4. Классификация нового объекта** $x = (13,\; 42)$ при $\lambda_{-1} = \lambda_{+1} = 1$.

Общий множитель (одинаков для обоих классов, так как дисперсии совпали):

$$\ln(0.5) + \ln\frac{1}{2\pi\sqrt{8/3}\cdot\sqrt{50/3}} = -0.693 + \ln\frac{3}{40\pi} \approx -0.693 - 3.735 = -4.428$$

Для $y = -1$:

$$a(13,\,42 \mid y=-1) = -4.428 - \frac{(13-20)^2}{2\cdot 8/3} - \frac{(42-30)^2}{2\cdot 50/3} = -4.428 - \frac{49\cdot 3}{16} - \frac{144\cdot 3}{100} \approx -4.428 - 9.188 - 4.320 = -17.936$$

Для $y = +1$:

$$a(13,\,42 \mid y=+1) = -4.428 - \frac{(13-10)^2}{2\cdot 8/3} - \frac{(42-50)^2}{2\cdot 50/3} = -4.428 - \frac{9\cdot 3}{16} - \frac{64\cdot 3}{100} \approx -4.428 - 1.688 - 1.920 = -8.036$$

Так как $a(y=+1) = -8.036 > a(y=-1) = -17.936$, классифицируем как $y^* = +1$. Объект $(13,\,42)$ ближе к центру кластера $+1$ ($(10,\,50)$), чем к центру $-1$ ($(20,\,30)$).
