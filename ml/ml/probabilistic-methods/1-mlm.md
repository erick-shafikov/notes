# Логистическая регрессия, Метод максимального правдоподобия

При бинарной классификации вместо жесткого ответа используется вероятность принадлежности к классу. Пусть $P(y | x, \omega)$ — вероятность принадлежности объекта $x$ классу $y$.

В логистической регрессии эта вероятность моделируется сигмоидой:

$$\sigma(t) = \frac{1}{1 + e^{-t}}$$

Она принимает любое вещественное число и возвращает значение в $(0, 1)$, что естественно интерпретировать как вероятность. Соответствующая функция потерь на одном объекте:

$$L(M) = -\log \sigma(M) = \log(1 + e^{-M})$$

Это одно и то же выражение в двух формах — логистическая функция потерь штрафует за отрицательный отступ (ошибку) и слабо штрафует за большой положительный отступ (уверенную правильную классификацию).

Задача — найти такие параметры $\omega$, которые максимизируют правдоподобие выборки:

$$\omega^* = \arg\max \prod_{i=1}^l P(y_i \mid x_i, \omega)$$

Предполагается независимость объектов, тогда:

$$P(y_1, y_2, \ldots, y_l \mid x_1, x_2, \ldots, x_l, \omega) = \prod_{i=1}^l P(y_i \mid x_i, \omega)$$

Для удобства оптимизации переходят к логарифму: произведение большого числа вероятностей (каждая из которых меньше 1) быстро стремится к нулю при росте выборки, что приводит к численному underflow. Логарифм превращает произведение в сумму и не меняет положение максимума, так как $\log$ — монотонно возрастающая функция:

$$\log L(\omega) = \sum_{i=1}^l \log P(y_i \mid x_i, \omega) \to \max$$

Эквивалентно можно минимизировать функцию потерь:

$$Q(\omega) = -\sum_{i=1}^l \log P(y_i \mid x_i, \omega) \to \min$$

Таким образом, вероятностная задача максимизации правдоподобия сводится к задаче минимизации функции потерь.

**Логистическая регрессия.** Классы кодируются как $y \in \{-1, +1\}$. Для обоих классов сразу можно записать:

$$P(y \mid x, \omega) = \sigma(y \cdot \langle \omega, x \rangle) = \frac{1}{1 + e^{-y \langle \omega, x \rangle}}$$

Это работает потому что при $y = +1$ получаем $\sigma(\langle \omega, x \rangle)$, а при $y = -1$ — $\sigma(-\langle \omega, x \rangle) = 1 - \sigma(\langle \omega, x \rangle)$.

Вводим отступ $M_i = y_i \cdot \langle \omega, x_i \rangle$ — это мера того, насколько уверенно и правильно модель классифицирует объект: большой положительный $M_i$ означает правильную классификацию с запасом, отрицательный — ошибку.

Тогда:

$$P(y_i \mid x_i, \omega) = \frac{1}{1 + e^{-M_i}}$$

Подставляем в функцию потерь:

$$Q(\omega) = -\sum_{i=1}^l \log \frac{1}{1 + e^{-M_i}} = \sum_{i=1}^l \log(1 + e^{-M_i}) \to \min$$

Вероятность класса задается сигмоидой:
$P(y = 1 | x, ω) = 1 / (1 + e^{-<ω, x>})$

## Многоклассовая классификация: Multinomial Logistic Regression

Бинарная логистическая регрессия моделирует одну вероятность $p = P(y=1 \mid x)$, а вероятность второго класса получается как $1 - p$. При $K > 2$ классах нужно одновременно оценить $K$ вероятностей с условием $\sum_{k=1}^K P(y=k \mid x) = 1$. Для этого вводится отдельный вектор весов $w_k$ для каждого класса, а вероятности получаются через **softmax**:

$$P(y = k \mid x, W) = \frac{\exp(\langle w_k, x \rangle)}{\sum_{j=1}^{K} \exp(\langle w_j, x \rangle)}$$

где $W = \{w_1, \ldots, w_K\}$ — матрица всех весов, $\langle w_k, x \rangle$ — линейный скор класса $k$. Знаменатель нормирует все $K$ экспонент в единицу, гарантируя корректные вероятности.

Softmax — прямое обобщение сигмоиды: при $K=2$ оба выражения эквивалентны. Скор $\langle w_k, x \rangle$ может быть любым вещественным числом, softmax переводит вектор скоров в вероятностное распределение по классам.

**Вывод функции потерь.** Правдоподобие для одного объекта при истинном классе $y_i = k$:

$$P(y_i \mid x_i, W) = \prod_{k=1}^{K} P(y=k \mid x_i, W)^{\mathbb{1}[y_i = k]}$$

где $\mathbb{1}[y_i = k]$ выбирает только вероятность истинного класса. Логарифмируя и меняя знак, получаем **categorical cross-entropy**:

$$Q(W) = -\sum_{i=1}^{l} \sum_{k=1}^{K} \mathbb{1}[y_i = k] \cdot \log P(y=k \mid x_i, W) = -\sum_{i=1}^{l} \log P(y = y_i \mid x_i, W) \to \min$$

По сути каждый объект вносит в потерю только один ненулевой член — логарифм вероятности истинного класса. Чем увереннее модель предсказывает правильный класс, тем меньше потеря.

**Градиент** по весам класса $k$:

$$\frac{\partial Q}{\partial w_k} = \sum_{i=1}^{l} \bigl(P(y=k \mid x_i, W) - \mathbb{1}[y_i = k]\bigr)\, x_i$$

Структура та же, что у бинарного случая: разность между предсказанной вероятностью и истинной меткой, взвешенная по признакам. Если модель уверена и права — $P(y=k \mid x_i) \approx 1$, $\mathbb{1}[y_i=k] = 1$ — градиент близок к нулю. Если ошибается — градиент большой и корректирует веса в нужную сторону.

**Связь с бинарным случаем.** При $K=2$ один из весов можно зафиксировать нулевым ($w_2 = 0$) без потери общности, тогда:

$$P(y=1 \mid x) = \frac{e^{\langle w_1, x \rangle}}{e^{\langle w_1, x \rangle} + 1} = \sigma(\langle w_1, x \rangle)$$

что в точности воспроизводит бинарную логистическую регрессию.
