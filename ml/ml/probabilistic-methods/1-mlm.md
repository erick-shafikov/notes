# Логистическая регрессия, Метод максимального правдоподобия

При бинарной классификации вместо жесткого ответа используется вероятность принадлежности к классу. Пусть $P(y | x, \omega)$ — вероятность принадлежности объекта $x$ классу $y$.

**Логистическая регрессия** — линейная модель, которая предсказывает именно эту вероятность. Линейная комбинация признаков $\langle \omega, x \rangle$ принимает любое вещественное значение, но вероятность должна лежать в $(0, 1)$. Поэтому линейный скор пропускают через функцию, которая "сжимает" $\mathbb{R}$ в $(0, 1)$ — **сигмоиду**.

В логистической регрессии эта вероятность моделируется сигмоидой:

$$\sigma(t) = \frac{1}{1 + e^{-t}}$$

где $t = \langle \omega, x \rangle$ — линейный скор объекта. Сигмоида принимает любое вещественное число и возвращает значение в $(0, 1)$, что естественно интерпретировать как вероятность. Соответствующая функция потерь на одном объекте:

$$L(M) = -\log \sigma(M) = \log(1 + e^{-M})$$

Это одно и то же выражение в двух формах — логистическая функция потерь штрафует за отрицательный отступ (ошибку) и слабо штрафует за большой положительный отступ (уверенную правильную классификацию).

Связь между $\sigma(t)$ и $L(M)$: аргументы у них разные. $\sigma$ получает сырой скор $t = \langle \omega, x \rangle$ и предсказывает вероятность, не зная про истинную метку. $L$ получает **отступ** $M = y \cdot \langle \omega, x \rangle$, который уже учитывает правильность: при верном ответе $M > 0$, при ошибке $M < 0$. Потеря $-\log \sigma(M)$ штрафует именно за уверенность в неправильном направлении: если $M \gg 0$ — $\sigma(M) \approx 1$, потеря $\approx 0$; если $M < 0$ — $\sigma(M) < 0.5$, потеря большая.

Задача — найти такие параметры $\omega$, которые максимизируют правдоподобие выборки:

$$\omega^* = \arg\max \prod_{i=1}^l P(y_i \mid x_i, \omega)$$

Предполагается независимость объектов, тогда:

$$P(y_1, y_2, \ldots, y_l \mid x_1, x_2, \ldots, x_l, \omega) = \prod_{i=1}^l P(y_i \mid x_i, \omega)$$

Для удобства оптимизации переходят к логарифму: произведение большого числа вероятностей (каждая из которых меньше 1) быстро стремится к нулю при росте выборки, что приводит к численному underflow. Логарифм превращает произведение в сумму и не меняет положение максимума, так как $\log$ — монотонно возрастающая функция:

$$\log L(\omega) = \sum_{i=1}^l \log P(y_i \mid x_i, \omega) \to \max$$

Эквивалентно можно минимизировать функцию потерь:

$$Q(\omega) = -\sum_{i=1}^l \log P(y_i \mid x_i, \omega) \to \min$$

Таким образом, вероятностная задача максимизации правдоподобия сводится к задаче минимизации функции потерь.

**Логистическая регрессия.** Классы кодируются как $y \in \{-1, +1\}$. Для обоих классов сразу можно записать:

$$P(y \mid x, \omega) = \sigma(y \cdot \langle \omega, x \rangle) = \frac{1}{1 + e^{-y \langle \omega, x \rangle}}$$

Это работает потому что при $y = +1$ получаем $\sigma(\langle \omega, x \rangle)$, а при $y = -1$ — $\sigma(-\langle \omega, x \rangle) = 1 - \sigma(\langle \omega, x \rangle)$.

Вводим отступ $M_i = y_i \cdot \langle \omega, x_i \rangle$ — это мера того, насколько уверенно и правильно модель классифицирует объект: большой положительный $M_i$ означает правильную классификацию с запасом, отрицательный — ошибку.

Тогда:

$$P(y_i \mid x_i, \omega) = \frac{1}{1 + e^{-M_i}}$$

Подставляем в функцию потерь:

$$Q(\omega) = -\sum_{i=1}^l \log \frac{1}{1 + e^{-M_i}} = \sum_{i=1}^l \log(1 + e^{-M_i}) \to \min$$

Вероятность класса задается сигмоидой:
$P(y = 1 | x, ω) = 1 / (1 + e^{-<ω, x>})$

## Многоклассовая классификация: Multinomial Logistic Regression

Бинарная логистическая регрессия моделирует одну вероятность $p = P(y=1 \mid x)$, а вероятность второго класса получается как $1 - p$. При $K > 2$ классах нужно одновременно оценить $K$ вероятностей с условием $\sum_{k=1}^K P(y=k \mid x) = 1$. Для этого вводится отдельный вектор весов $w_k$ для каждого класса, а вероятности получаются через **softmax**:

$$P(y = k \mid x, W) = \frac{\exp(\langle w_k, x \rangle)}{\sum_{j=1}^{K} \exp(\langle w_j, x \rangle)}$$

где $W = \{w_1, \ldots, w_K\}$ — матрица всех весов, $\langle w_k, x \rangle$ — линейный скор класса $k$. Знаменатель нормирует все $K$ экспонент в единицу, гарантируя корректные вероятности.

Softmax — прямое обобщение сигмоиды: при $K=2$ оба выражения эквивалентны. Скор $\langle w_k, x \rangle$ может быть любым вещественным числом, softmax переводит вектор скоров в вероятностное распределение по классам.

**Вывод функции потерь.** Softmax даёт вероятность конкретного класса $k$, но для функции потерь нужно записать вероятность истинного класса объекта $i$ в общем виде — не зная заранее, какой класс истинный. Для этого используется произведение по всем классам с индикаторной функцией в показателе степени:

$$P(y_i \mid x_i, W) = \prod_{k=1}^{K} P(y=k \mid x_i, W)^{\mathbb{1}[y_i = k]}$$

где $\mathbb{1}[y_i = k]$ — индикаторная функция: равна $1$ если $k$ совпадает с истинным классом $y_i$, и $0$ иначе. Для каждого $k \neq y_i$ показатель степени равен $0$, множитель превращается в $1$ и не влияет на произведение. Для $k = y_i$ показатель равен $1$, остаётся $P(y = y_i \mid x_i, W)$. Это математический способ выбрать из softmax вероятность именно истинного класса. Логарифмируя и меняя знак, получаем **categorical cross-entropy**.

Логарифм произведения по $k$ превращается в сумму по $k$, степени $\mathbb{1}[y_i=k]$ спускаются вниз как множители — это применяется к каждому объекту $i$:

$$\log P(y_i \mid x_i, W) = \log \prod_{k=1}^{K} P(y=k \mid x_i, W)^{\mathbb{1}[y_i=k]} = \sum_{k=1}^{K} \mathbb{1}[y_i=k] \cdot \log P(y=k \mid x_i, W)$$

Суммируя по всем объектам и меняя знак:

$$Q(W) = -\sum_{i=1}^{l} \sum_{k=1}^{K} \mathbb{1}[y_i = k] \cdot \log P(y=k \mid x_i, W) = -\sum_{i=1}^{l} \log P(y = y_i \mid x_i, W) \to \min$$

По сути каждый объект вносит в потерю только один ненулевой член — логарифм вероятности истинного класса. Чем увереннее модель предсказывает правильный класс, тем меньше потеря.

**Числовой пример.** Softmax — это модель: принимает скоры и выдаёт вероятности классов. Произведение-трюк ниже — для функции потерь: из всех $K$ вероятностей нужно вытащить только одну — истинного класса.

$K = 3$ класса, один объект $x_i$ с истинным классом $y_i = 2$. Линейные скоры:

$$\langle w_1, x_i \rangle = 1.0, \quad \langle w_2, x_i \rangle = 3.0, \quad \langle w_3, x_i \rangle = 0.5$$

**Шаг 1. Softmax** — переводим скоры в вероятности:

$$e^{1.0} \approx 2.718, \quad e^{3.0} \approx 20.086, \quad e^{0.5} \approx 1.649, \quad \text{сумма} \approx 24.453$$

$$P(y=1 \mid x_i) \approx \frac{2.718}{24.453} \approx 0.111, \quad P(y=2 \mid x_i) \approx \frac{20.086}{24.453} \approx 0.821, \quad P(y=3 \mid x_i) \approx \frac{1.649}{24.453} \approx 0.067$$

**Шаг 2. Произведение-трюк** — $y_i = 2$, поэтому $\mathbb{1}[y_i = k]$ равна $1$ только при $k=2$:

$$\prod_{k=1}^{3} P(y=k \mid x_i)^{\mathbb{1}[y_i=k]} = \underbrace{0.111^0}_{k=1} \cdot \underbrace{0.821^1}_{k=2} \cdot \underbrace{0.067^0}_{k=3} = 1 \cdot 0.821 \cdot 1 = 0.821$$

**Шаг 3. Потеря** для этого объекта:

$$-\log P(y = y_i \mid x_i) = -\log 0.821 \approx 0.197$$

Что совпадает с прямым раскрытием через softmax:

$$-\log P(y=2 \mid x_i) = -\bigl(\langle w_2, x_i \rangle - \log \textstyle\sum_j e^{\langle w_j, x_i \rangle}\bigr) = -(3.0 - \log 24.453) \approx -(3.0 - 3.197) = 0.197$$

**Градиент** по весам класса $k$:

$$\frac{\partial Q}{\partial w_k} = \sum_{i=1}^{l} \bigl(P(y=k \mid x_i, W) - \mathbb{1}[y_i = k]\bigr)\, x_i$$

Структура та же, что у бинарного случая: разность между предсказанной вероятностью и истинной меткой, взвешенная по признакам. Если модель уверена и права — $P(y=k \mid x_i) \approx 1$, $\mathbb{1}[y_i=k] = 1$ — градиент близок к нулю. Если ошибается — градиент большой и корректирует веса в нужную сторону.

**Связь с бинарным случаем.** При $K=2$ один из весов можно зафиксировать нулевым ($w_2 = 0$) без потери общности, тогда:

$$P(y=1 \mid x) = \frac{e^{\langle w_1, x \rangle}}{e^{\langle w_1, x \rangle} + 1} = \sigma(\langle w_1, x \rangle)$$

что в точности воспроизводит бинарную логистическую регрессию.

## Итог

Метод максимального правдоподобия даёт единый взгляд на все рассмотренные модели: мы хотим, чтобы модель как можно выше оценивала вероятность наблюдённых меток, — это и есть максимизация правдоподобия.

Ключевые роли разделены чётко:

- **Модель** (сигмоида / softmax) выдаёт вероятность принадлежности к классу $P(y \mid x, \omega) \in (0, 1)$.
- **Функция потерь** принимает эту вероятность и вычисляет штраф: $L = -\log P(y \mid x, \omega)$.

Функция потерь не выдаёт вероятность — она измеряет, насколько плохо модель предсказала правильный класс. Чем выше вероятность истинного класса, тем ближе штраф к нулю.

Цепочка эквивалентностей:

$$\max_\omega \prod_{i=1}^l P(y_i \mid x_i, \omega) \;\Longleftrightarrow\; \max_\omega \sum_{i=1}^l \log P(y_i \mid x_i, \omega) \;\Longleftrightarrow\; \min_\omega \underbrace{-\sum_{i=1}^l \log P(y_i \mid x_i, \omega)}_{\text{cross-entropy}}$$

Бинарный случай даёт логистическую функцию потерь $\log(1 + e^{-M_i})$, многоклассовый — categorical cross-entropy. Градиент в обоих случаях имеет одну и ту же структуру: разность между предсказанной вероятностью и истинной меткой.
