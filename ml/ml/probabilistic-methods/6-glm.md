# Обобщённые линейные модели (GLM)

Обычная линейная регрессия предполагает, что ответ $y$ непрерывен и его ошибки нормально распределены: $y \sim \mathcal{N}(\mu, \sigma^2)$. На практике это выполняется редко — бинарные ответы (да/нет), счётные данные (число кликов, число ошибок), вероятности требуют других распределений. **Generalized Linear Models (GLM)** — это фреймворк, который сохраняет линейную структуру модели, но позволяет гибко выбирать распределение ответа и способ связи линейной части с его средним.

## Три компоненты GLM

Любая GLM задаётся тремя элементами.

**1. Распределение** целевой переменной $y \mid x$ из экспоненциального семейства. Выбор определяет природу задачи:

| Распределение | Область $y$           | Пример задачи                      |
| ------------- | --------------------- | ---------------------------------- |
| Gaussian      | $\mathbb{R}$          | регрессия цены, температуры        |
| Bernoulli     | $\{0, 1\}$            | бинарная классификация             |
| Poisson       | $\{0, 1, 2, \ldots\}$ | число событий за период            |
| Gamma         | $\mathbb{R}_{>0}$     | время до отказа, страховые выплаты |
| Binomial      | $\{0,\ldots,n\}$      | число успехов из $n$ попыток       |

**2. Линейный предиктор** — стандартная линейная комбинация признаков:

$$\eta = w^T x + b$$

где $w$ — вектор весов, $b$ — свободный член.

**3. Функция связи** $g$ — монотонная дифференцируемая функция, которая связывает математическое ожидание ответа $\mu = \mathbb{E}[y \mid x]$ с линейным предиктором:

$$g(\mu) = \eta = w^T x + b$$

Обратная к ней функция $g^{-1}$ называется **response function** и даёт прогноз:

$$\mu = g^{-1}(w^T x + b)$$

Интуиция функции связи: линейный предиктор $\eta$ принимает любые вещественные значения $(-\infty, +\infty)$, тогда как $\mu$ может быть ограничено — вероятность лежит в $[0,1]$, интенсивность событий $> 0$. Функция связи «растягивает» ограниченную область значений $\mu$ на всю числовую прямую.

## Сквозная логика GLM

Все три компоненты работают в одной цепочке — от признаков до минимизируемой потери:

```
η = w^T x + b          ← линейный предиктор, любое число из (-∞, +∞)
         ↓  g⁻¹        ← обратная функция связи отображает η в область μ
μ = g⁻¹(η)             ← прогноз среднего: напр. вероятность в (0,1) или λ > 0
         ↓             ← выбранное распределение задаёт правдоподобие
L(w) = ∏ p(yᵢ | μᵢ)   ← насколько хорошо μᵢ объясняет наблюдённые yᵢ
         ↓  −log        ← логарифмируем и меняем знак
−log L(w) → min        ← функция потерь; минимизируем по w
```

Важно: обратная функция связи $g^{-1}$ даёт **прогноз** $\mu$, а не потерю. Потеря рождается из распределения — из того, насколько $\mu$ объясняет реальные $y_i$. Именно поэтому кросс-энтропия при Bernoulli и MSE при Gaussian — не произвольные выборы, а прямые следствия MLE для соответствующих распределений.

Диапазон $\mu$ зависит от задачи и определяется распределением, а не фиксирован как $[0, 1]$:

| Распределение | Диапазон $\mu$ | Что ограничивает |
|---|---|---|
| Bernoulli | $(0, 1)$ | вероятность |
| Poisson | $(0, +\infty)$ | интенсивность событий |
| Gaussian | $(-\infty, +\infty)$ | нет ограничений |
| Gamma | $(0, +\infty)$ | положительные величины |

## Экспоненциальное семейство распределений

GLM работают с распределениями, которые можно записать в стандартной форме:

$$p(y \mid \theta) = h(y)\exp\!\bigl(\eta(\theta)\, T(y) - A(\theta)\bigr)$$

Каждая составляющая играет свою роль.

**$\theta$** — параметр распределения в привычном смысле: для Gaussian это $(\mu, \sigma^2)$, для Bernoulli это $p$, для Poisson это $\lambda$. Именно $\theta$ определяет конкретный экземпляр распределения внутри семейства.

**$\eta(\theta)$** — **натуральный параметр** (natural parameter). Это перепараметризация $\theta$, в которой формула распределения принимает стандартный экспоненциальный вид. Например, для Bernoulli $\eta = \log\frac{p}{1-p}$ (логит), для Poisson $\eta = \log\lambda$. Именно $\eta$ линейно зависит от признаков в GLM: $\eta = w^T x + b$.

**$T(y)$** — **достаточная статистика**. Это функция наблюдения $y$, которая содержит всю информацию о параметре $\theta$, присутствующую в данных. Для большинства простых распределений $T(y) = y$ — само наблюдение и является достаточной статистикой. Произведение $\eta \cdot T(y)$ — это «взаимодействие» параметра модели и наблюдения внутри показателя экспоненты.

**$A(\theta)$** — **log-partition function** (логарифмическая нормировочная константа). Обеспечивает, чтобы $p(y \mid \theta)$ суммировалась (или интегрировалась) в единицу:

$$A(\theta) = \log \int h(y)\exp\!\bigl(\eta(\theta)\, T(y)\bigr)\, dy$$

У неё есть важное свойство: её производные по $\eta$ дают моменты распределения:

$$\frac{\partial A}{\partial \eta} = \mathbb{E}[T(y)] = \mu, \qquad \frac{\partial^2 A}{\partial \eta^2} = \mathrm{Var}[T(y)]$$

Это делает $A(\theta)$ центральным объектом при вычислении градиентов в задаче обучения.

**$h(y)$** — **базовая мера**. Множитель, не зависящий от параметров модели — он задаёт «фоновую» форму распределения по $y$ до учёта параметра $\theta$. Для непрерывных распределений это может быть $\frac{1}{\sqrt{2\pi}}$, для дискретных — нормировочный коэффициент вроде $\binom{n}{y}$.

Конкретные примеры:

| Распределение | $\eta(\theta)$ | $T(y)$ | $A(\theta)$ | $h(y)$ |
|---|---|---|---|---|
| Gaussian ($\mu$, $\sigma^2$ фиксирована) | $\mu / \sigma^2$ | $y$ | $\mu^2 / 2\sigma^2$ | $\frac{1}{\sqrt{2\pi}\sigma}e^{-y^2/2\sigma^2}$ |
| Bernoulli | $\log\frac{p}{1-p}$ | $y$ | $\log(1 + e^\eta)$ | $1$ |
| Poisson | $\log \lambda$ | $y$ | $\lambda = e^\eta$ | $\frac{1}{y!}$ |

В это семейство входят Gaussian, Bernoulli, Binomial, Poisson, Gamma, Beta и другие. Обычная линейная регрессия, логистическая и регрессия Пуассона — все частные случаи одного фреймворка.

## Канонические функции связи

Для каждого распределения существует **каноническая** функция связи — она получается из натурального параметра $\eta(\mu)$ напрямую и упрощает математику обучения: градиент log-likelihood принимает особенно чистую форму.

| Распределение | Каноническая связь $g(\mu)$               | Область $\mu$  |
| ------------- | ----------------------------------------- | -------------- |
| Gaussian      | $g(\mu) = \mu$ (identity)                 | $\mathbb{R}$   |
| Bernoulli     | $g(\mu) = \log\dfrac{\mu}{1-\mu}$ (logit) | $(0, 1)$       |
| Poisson       | $g(\mu) = \log \mu$ (log)                 | $(0, +\infty)$ |
| Gamma         | $g(\mu) = -1/\mu$ (inverse)               | $(0, +\infty)$ |

## Логистическая регрессия как GLM

**Задача:** бинарная классификация $y \in \{0, 1\}$.

**Распределение:** $y \mid x \sim \mathrm{Bernoulli}(p)$, где $p = P(y=1 \mid x)$.

**Каноническая связь** — логит:

$$g(p) = \log \frac{p}{1 - p} = w^T x + b$$

где $\log\frac{p}{1-p}$ — логарифм отношения шансов (log-odds): при $p = 0.5$ равно $0$, при $p \to 1$ уходит в $+\infty$, при $p \to 0$ — в $-\infty$, что покрывает всю числовую прямую.

Обращая функцию связи, получаем сигмоиду:

$$p = g^{-1}(\eta) = \frac{1}{1 + e^{-(w^T x + b)}}$$

**Почему функция потерь — кросс-энтропия.** Это не произвольный выбор, а прямое следствие принципа максимального правдоподобия. Правдоподобие выборки при $y_i \in \{0,1\}$:

$$L(w) = \prod_{i=1}^l p_i^{y_i}(1 - p_i)^{1 - y_i}$$

Логарифмируя и меняя знак:

$$-\log L(w) = -\sum_{i=1}^l \bigl[y_i \log p_i + (1 - y_i)\log(1 - p_i)\bigr] \to \min$$

Это в точности **кросс-энтропийная функция потерь**. Таким образом, минимизация кросс-энтропии эквивалентна максимизации правдоподобия при предположении Bernoulli-распределения.

## Регрессия Пуассона как GLM

**Задача:** предсказание числа событий $y \in \{0, 1, 2, \ldots\}$ — количество заказов, кликов, ошибок в коде.

**Распределение:** $y \mid x \sim \mathrm{Poisson}(\lambda)$, где $\lambda = \mathbb{E}[y \mid x] > 0$.

**Каноническая связь** — логарифм:

$$g(\lambda) = \log \lambda = w^T x + b$$

Обратное преобразование гарантирует положительность прогноза:

$$\lambda = e^{w^T x + b} > 0$$

Log-likelihood Пуассона:

$$\log L(w) = \sum_{i=1}^l \bigl(y_i \log \lambda_i - \lambda_i - \log y_i!\bigr)$$

Минимизация $-\log L(w)$ по $w$ — это и есть задача обучения регрессии Пуассона.

## Обучение GLM: метод максимального правдоподобия

Все GLM обучаются одним методом — **MLE**:

$$w^* = \arg\max_w \sum_{i=1}^l \log p(y_i \mid x_i, w)$$

Благодаря свойствам экспоненциального семейства log-likelihood является **вогнутой** функцией от $w$ — у задачи оптимизации единственный глобальный максимум, и к нему можно прийти градиентным методом. Для канонической функции связи градиент имеет особенно чистую форму:

$$\nabla_w \log L = \sum_{i=1}^l (y_i - \mu_i)\, x_i$$

что совпадает со стандартным градиентом MSE-потери в линейной регрессии — разность «истинное значение минус прогноз», взвешенная по признакам.

## Линейная регрессия — частный случай GLM

При $y \sim \mathcal{N}(\mu, \sigma^2)$ и identity link $g(\mu) = \mu$ модель принимает вид:

$$\mu = w^T x + b, \qquad y = w^T x + b + \varepsilon, \quad \varepsilon \sim \mathcal{N}(0, \sigma^2)$$

Запишем правдоподобие одного наблюдения через плотность нормального распределения:

$$p(y_i \mid x_i, w) = \frac{1}{\sqrt{2\pi}\,\sigma} \exp\!\left(-\frac{(y_i - \mu_i)^2}{2\sigma^2}\right)$$

Логарифмируя и суммируя по всей выборке, получаем log-likelihood:

$$\log L(w) = -\frac{l}{2}\log(2\pi\sigma^2) - \frac{1}{2\sigma^2}\sum_{i=1}^l (y_i - \mu_i)^2$$

Первое слагаемое не зависит от $w$, поэтому максимизация $\log L$ по $w$ сводится к минимизации второго:

$$-\log L(w) \propto \sum_{i=1}^l (y_i - w^T x_i - b)^2$$

Это в точности MSE. Метод наименьших квадратов, таким образом, — не эвристика, а MLE под предположением о нормальности ошибок: если $\varepsilon \sim \mathcal{N}(0, \sigma^2)$, то МНК даёт наиболее правдоподобные параметры.

## Связь с нейронными сетями

Один нейрон с активационной функцией структурно совпадает с GLM: линейная комбинация входов передаётся через нелинейность. Разница — в том, что в GLM нелинейность задаётся обратной функцией связи $g^{-1}$ и выбирается исходя из распределения ответа, а не произвольно:

| Нейрон         | Аналог в GLM                        |
| -------------- | ----------------------------------- |
| Sigmoid-нейрон | Логистическая регрессия (Bernoulli) |
| Softmax-слой   | Мультиномиальная GLM                |
| Линейный выход | Линейная регрессия (Gaussian)       |
| Exp-выход      | Регрессия Пуассона                  |

Глубокая сеть можно рассматривать как GLM, у которого признаки $x$ заменены на нелинейные представления из скрытых слоёв.

## Ограничения GLM

GLM предполагают линейность в пространстве признаков после применения функции связи. Для сложных нелинейных зависимостей с взаимодействиями признаков применяют:

- градиентный бустинг (XGBoost, LightGBM)
- случайный лес
- нейронные сети

При этом GLM остаются предпочтительными там, где важна интерпретируемость коэффициентов: медицина, кредитный скоринг, социологические исследования.
