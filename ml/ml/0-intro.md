# Постановка задачи ML

$X^l = {(x_i, y_i)}{i=1}^l$ — размеченная обучающая выборка
$F = {f_j(x)}{j=1}^n$ — пространство признаков

$f_j(x_i)$ — некий признак
$a(x)$ — формируемая модель, решающая функция (decision function)
$y_i = y(x_i)$ — истинный ответ

$a(x) = g(x, θ)$ — параметрическая модель, где
$θ = [θ_1, θ_2, ..., θ_n]$

На примере линейной функции

$y = k * x + b + ε$, где $ε$ гауссовский шум
$g(x, k, b) = k * x + b$
$θ = [k, b]$

Критерий качества

$L(a, x) = |a(x) - y(x)|$ — абсолютная ошибка
$L(a, x) = (a(x) - y(x)) ^ 2$ — квадратичная ошибка

$Q(a, X^l) = 1 / l * Σ_{i=1}^l L(a, x_i)$ — средний эмпирический риск

Для квадратичной функции потерь и линейной модели

$Q = 1 / l * Σ_{i=1}^l (k * x_i + b - y_i) ^ 2$ — цель минимизации

Задача сводится к методу наименьших квадратов (МНК)

$∂Q/∂k = 2/l Σ_{i=1}^l (k * x_i + b - y_i) * x_i = 0$

$∂Q/∂b = 2/l Σ_{i=1}^l (k * x_i + b - y_i) = 0$

Общая задача — поиск модели, описывающей зависимость в данных
Часто сводится к задаче параметрической оптимизации
Для нахождения коэффициентов минимизируется средний эмпирический риск

# задачи

Задачи подразделяются на:

задачи регрессии — прогноз
задачи классификации — определение объектов
задачи ранжирования — обработка запросов
обучающая выборка

ОВ представлена в виде векторов

$x_i = [x_{i1}, x_{i2}, ..., x_{in}]^T$

$X = [x_1, x_2, ... x_l] =  \begin{bmatrix} x_{11}, x_{12},...x_{1n} \\ ... \\ x_{l1}, x_{l2},...x_{ln}\end{bmatrix}$

$n$ — число признаков
$l$ — число объектов
входные данные, целевые значения

$Y = [y_1, y_2, ..., y_l]$

При классификации

$y ∈ {-1, 1}$ — бинарная классификация

$y ∈ {1, 2, 3, ..., m}$ — многоклассовая классификация

$y ∈ {0, 1}^M$ — многоклассовая классификация с пересекающимися классами, $M$ — размерность

$y ∈ R$ — одномерный случай

$y ∈ R^m$ — многомерный случай

$X^l = {(x_i, y_i)}_{i=1}^l$ — размеченная обучающая выборка

Не просто пиксели:

- вторичные признаки
- feature extraction
- feature engineering

$[f_1(x), f_2(x), ..., f_n(x)]$ — признаки

$F =\begin{bmatrix}f_1(x_1), ... f_n(x_1) \\ ... \\ f_1(x_l)  , ...  f_n(x_l)\end{bmatrix}$
