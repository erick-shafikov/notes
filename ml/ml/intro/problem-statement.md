```python
f1(xi) # некий признак
a = a(xi) # формируемая модель, решающая функция decision function
yi = y(xi) # некий результат работы алгоритма

a(x) = g(x, θ) # тогда функцию a параметрическая функция, где
θ = [θ1,θ2,...,θn]
```

На примере линейной функции

```python
y = k * x + b + ε # ε - некий Гауссовский шум
g(x, k,b) = k * x + b # где
θ = [k, b]
```

Критерий качества

```python
L(a, x) = abs(a(x) - g(x)) # абсолютная ошибка, но она не подходит так как не увеличивается при ошибке
L(a, x) = (a(x) - g(x)) ^ 2 # квадратичная
Q(a, x^l) =1 / l * sum(L(a, xi)) # должно минимизироваться

# для квадратичной функции потерь и линейной моделью

Q = 1 / l * sum(k * xi + b - y(xi)) ^ 2 # основная цель минимизировать

# задача сводится к МНК

{
dQ(a, x) / dx = sum(k * xi + b - y(xi)) * xi = 0;
dQ(a, x) / db = sum(k * xi + b - y(xi)) = 0;
}
```

- Общая задача - поиск модели описывающую природу процесса
- Часто сводится к задача параметрической оптимизации
- Для нахождения коэффициентов выводится средний эмпирический риск
