# обучающая выборка

ОВ представлена в виде векторов

```python
xi = [
  xi1,
  xi2,
  ...
  xi1,
  ] = [xi1,xi2,...xi1,].T

X = [x1, x2,...xn] = [
  x11,x12,...x1n, # признаки
  ...
  xl1,xl2,...xln,
  ]

- n - число признаков
- l - число объектов

# входные данные, целевые значения
Y = [y1, y2, ..., yn]
```

При классификации

```python
y = [-1, 1] # бинарная классификация
y = [1,2,3, ..., m] # m-классовая классификация
y = [0, 1] ^ M # m-классовая классификация c пересекающимися классами, M - размерность пространства

y ∈ R # одномерная
y ∈ R ^ m # m-мерная

X^l = {(xi,yi)^^i=1^l } # размеченная обучающая выборка
```
