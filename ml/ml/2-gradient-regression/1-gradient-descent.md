# Градиентны спуск

Нахождение минимума — производная (в одномерном случае). В многомерном случае используется градиент — вектор частных производных. Градиент показывает направление наискорейшего роста функции, поэтому для поиска минимума движение происходит в сторону антиградиента. Градиент — это значение производной (частных производных) в точке. Формула обновления: $x_{n+1} = x_n - λ * grad f(x_n)$, где $λ$ — шаг обучения. Смысл в том, что вместо решения уравнения $f'(x)=0$ (что может быть сложно или невозможно аналитически) используется итеративное приближение к минимуму.

```python
import numpy as np

def f(x):
    return x*x - 5*x + 5

def df(x):
    return 2*x - 5

N = 20     # число итераций
xx = 0      # начальное значение
lmd = 0.9  # шаг сходимости

for i in range(N):
    xx = xx - lmd*df(xx)    # изменение аргумента на текущей итерации
```

При расчете градиента также используют нормирование: $grad f(x, y, z, ...)={∂f/∂x, ∂f/∂y, ∂f/∂z, ...}$ — градиент функции. $|grad f(x,y,z,...)| = \sqrt{(∂f/∂x)^2 + (∂f/∂y)^2 + (∂f/∂z)^2 + ...}$ — его норма. Тогда нормированный градиент: $grad f / |grad f|$, и обновление может быть записано как $x_{n+1} = x_n - λ * (grad f(x_n) / |grad f(x_n)|)$. Таким образом движение всегда происходит в сторону антиградиента (уменьшения функции). Шаг λ может быть константным или изменяться по итерациям, например: $λ_n = 1 / min(n+1, m)$, где m — ограничение на минимальный шаг.

```python
import numpy as np


# исходная функция, которую нужно аппроксимировать моделью a(x)
def func(x):
    return 0.1 * x ** 2 - np.sin(x) + 5.


coord_x = np.arange(-5.0, 5.0, 0.1)  # значения по оси абсцисс [-5; 5] с шагом 0.1
coord_y = func(coord_x)  # значения функции по оси ординат

eta = np.array([0.1, 0.01, 0.001, 0.0001])  # шаг обучения для каждого параметра w0, w1, w2, w3
w = np.array([0., 0., 0., 0.])  # начальные значения параметров модели
N = 200  # число итераций градиентного алгоритма

X = np.power(coord_x.reshape(-1, 1), range(4)) # превращает одномерный x в матрицу полиномиальных признаков степени 3 (включая константу)

for _ in range(N):
    y_pred = X @ w
    w -= eta * 2 * ((y_pred - coord_y) * X.T).mean(1)

Q = np.square(y_pred - coord_y).mean()
```
