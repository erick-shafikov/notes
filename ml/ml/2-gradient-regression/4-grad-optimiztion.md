# оптимизация градиентных алгоритмов

# Метод импульсов (Momentum)

Метод используется для ускорения градиентного спуска и сглаживания колебаний. Идея в том, что к текущему шагу добавляется часть предыдущего шага, за счет чего накапливается “инерция” движения и алгоритм быстрее движется в одном направлении и меньше колеблется Формула итерации: $v_{n+1} = β * v_n + grad f(ω, x_n)$, $ω_{n+1} = ω_n - η * v_{n+1}$, где $v$ — накопленный вектор скорости (импульса), $β$ — коэффициент импульса (обычно 0.9), $η$ — шаг обучения. Смысл в том, что градиенты накапливаются во времени, и если направление сохраняется, шаги ускоряются, а если градиенты меняют знак, колебания сглаживаются. Экспоненциальное скользящее среднее градиента. Метод позволяет быстрее сходиться по “пологим” направлениям и уменьшает колебания в “крутых” направлениях по сравнению с обычным градиентным спуском.

# NAG

Метод стохастического градиентного спуска с импульсом Нестерова Nesterov's accelerated gradient. Пересчет идет с “заглядыванием вперед”, где градиент вычисляется не в текущей точке, а в точке с учетом импульса: $∇Q(ω - γ * v)$. Полная формула: $v_{n+1} = γ * v_n + η * ∇Q(ω_n - γ * v_n)$, $ω_{n+1} = ω_n - v_{n+1}$. Смысл в том, что сначала делается шаг вперед по импульсу, затем считается градиент, за счет чего корректируется направление движения заранее. Это уменьшает переезды минимума и делает сходимость более стабильной по сравнению с обычным momentum.

```python
import numpy as np
from numpy.polynomial import Polynomial


# исходная функция, которую нужно аппроксимировать моделью a(x)
def func(x):
    return -0.7 * x - 0.2 * x ** 2 + 0.05 * x ** 3 - 0.2 * np.cos(3 * x) + 2


def loss(x, y, w):
    return np.mean(np.square(x @ w - y))


def dqk(x, y, w, gamma, v):
    return ((w - gamma * v) @ x.T - y) @ x


coord_x = np.arange(-4.0, 6.0, 0.1)  # значения по оси абсцисс [-4; 6] с шагом 0.1
coord_y = func(coord_x)  # значения функции по оси ординат

sz = len(coord_x)  # количество значений функций (точек)
eta = np.array([0.1, 0.01, 0.001, 0.0001])  # шаг обучения для каждого параметра w0, w1, w2, w3
w = np.array([0., 0., 0., 0.])  # начальные значения параметров модели
N = 500  # число итераций алгоритма SGD
lm = 0.02  # значение параметра лямбда для вычисления скользящего экспоненциального среднего
batch_size = 20  # размер мини-батча (величина K = 20)
gamma = 0.8  # коэффициент гамма для вычисления импульсов Нестерова
v = np.zeros(len(w))  # начальное значение [0, 0, 0, 0]

np.random.seed(0)  # генерация одинаковых последовательностей псевдослучайных чисел

x_train = np.array([[1, x, x ** 2, x ** 3] for x in coord_x])
y_train = np.array(coord_y)

Qe = loss(x_train, y_train, w)  # начальное значение среднего эмпирического риска

for _ in range(N):
    k = np.random.randint(0, sz - batch_size - 1)
    rng = range(k, k + batch_size)
    x_range = x_train[rng]
    y_range = y_train[rng]
    v = gamma * v + (1 - gamma) * eta * 2 * dqk(x_range, y_range, w, gamma, v) / batch_size

    w -= v
    Qe = lm * loss(x_range, y_range, w) + (1 - lm) * Qe

Q = loss(x_train, y_train, w)

```

# RMSProp (running mean square)

Если параметров много, то значения частных производных могут сильно отличаться, из-за чего одни коэффициенты обновляются быстрее, чем другие. Градиент функции потерь представляет собой вектор частных производных по параметрам. Тогда можно применить метод, который адаптивно масштабирует шаг обучения для каждого параметра на основе накопленных квадратов градиентов.

Для этого вводится экспоненциальное скользящее среднее квадратов градиента: $G = α * G + (1 - α) * (∇L_i(ω) * ∇L_i(ω))$, где умножение поэлементное.

Пересчет весов идет по формуле: $w_{t + 1} = w_t - η * ∇L_i(ω_t) / (\sqrt{G} + ε)$, где $ε$ — маленькая константа для численной стабильности.

Смысл метода в том, что параметры с большими градиентами получают меньший шаг, а с маленькими — больший, что стабилизирует обучение и ускоряет сходимость.

# AdaDelta (adaptive learning rate)

$G = α * G + (1 - α) * (∇L_i(ω) * ∇L_i(ω))$, где умножение поэлементное

$E = α * E + (1 - α) * (δ * δ)$, где E — экспоненциальное среднее квадратов обновлений

$δ = - (√(E + ε) / (√(G + ε))) * ∇L_i(ω)$, где умножение поэлементное

$ω_{t + 1} = ω_t + δ$

Применяется экспоненциальное среднее квадратов градиентов и обновлений. Смысл метода в том, что шаг обновления масштабируется отношением накопленных обновлений к накопленным градиентам, за счет чего не требуется явный выбор шага обучения η.

# Adam (adaptive momentum)

$V = γ * V + (1 - γ) * ∇L_i(ω)$

$g = α * g + (1 - α) * (∇L_i(ω) * ∇L_i(ω))$, где умножение поэлементное

$\hat{V} = V / (1 - γ ^ t)$

$\hat{g} = g / (1 - α ^ t)$

$ω_{t + 1} = ω_t - η * \hat{V} / (\sqrt{\hat{g}} + ε)$

Используются экспоненциальные скользящие средние градиента и его квадратов с поправкой на смещение (bias correction). Смысл метода в том, что он сочетает накопление градиента (как в momentum) и адаптивное масштабирование шага (как в RMSProp), что делает обучение более стабильным и быстрым.

# Nadam (Nesterov adaptive momentum)

$V = γ * V + (1 - γ) * ∇L_i(ω)$

$g = α * g + (1 - α) * (∇L_i(ω) * ∇L_i(ω))$, где умножение поэлементное

$\hat{V} = V / (1 - γ ^ t)$

$\hat{g} = g / (1 - α ^ t)$

$\tilde{V} = γ * \hat{V} + (1 - γ) * ∇L_i(ω)$

$ω_{t + 1} = ω_t - η * \tilde{V} / (\sqrt{\hat{g}} + ε)$

Используются экспоненциальные скользящие средние градиента и его квадратов с поправкой на смещение, а также “заглядывание вперед” как в методе Нестерова. Смысл метода в том, что он сочетает идеи Adam (адаптивный шаг и накопление градиента) и NAG (lookahead), за счет чего достигается более стабильная и быстрая сходимость.

# Метод Левенберга-Марквардта (Levenberg–Marquardt)

Метод используется для решения задач нелинейной оптимизации и наименьших квадратов. Является комбинацией метода Гаусса-Ньютона и градиентного спуска: при малых значениях параметра ведет себя как метод Ньютона, при больших — как градиентный спуск.

$r(ω)$ — вектор остатков (ошибок), $J$ — матрица Якоби

$g = J^T r(ω)$ — градиент функции ошибки

$H ≈ J^T J$ — аппроксимация гессиана

$(J^T J + λ I) δ = J^T r(ω)$ — превращаем в диагональную матрицу

$ω_{t + 1} = ω_t - δ$

где $λ$ — параметр регуляризации (демпфирования), $I$ — единичная матрица.

Смысл метода в том, что он адаптивно переключается между быстрым методом второго порядка (Гаусса-Ньютона) и более устойчивым градиентным спуском. При уменьшении ошибки параметр $λ$ уменьшается (движение становится более “ньютоновским”), при увеличении ошибки — увеличивается (движение становится более стабильным).

№ AdamW (adaptive momentum with decoupled weight decay)

Метод является модификацией Adam, в котором регуляризация весов (weight decay) отделена от градиента. В обычном Adam L2-регуляризация добавляется к градиенту, что искажает адаптивное масштабирование, поэтому в AdamW штраф применяется отдельно.

$V = γ * V + (1 - γ) * ∇L_i(ω)$

$g = α * g + (1 - α) * (∇L_i(ω) * ∇L_i(ω))$, где умножение поэлементное

$\hat{V} = V / (1 - γ ^ t)$

$\hat{g} = g / (1 - α ^ t)$

$ω_{t + 1} = ω_t - η * \hat{V} / (\sqrt{\hat{g}} + ε) - η * λ * ω_t$

где $λ$ — коэффициент weight decay.

Смысл метода в том, что регуляризация применяется как отдельный шаг уменьшения весов, а не как часть градиента. Это позволяет сохранить корректную адаптацию шага (как в Adam) и одновременно контролировать величину весов. AdamW показывает лучшую обобщающую способность по сравнению с обычным Adam при наличии регуляризации.
