# Порождающие паттерны

Порождающие шаблоны описывают создание (instantiate) объекта или группы связанных объектов. Паттерны беспокоятся о гибком создании объектов без внесения в программу лишних зависимостей. Поражающие паттерны:

- [Фабричный метод](./factory-method.md)
- [Строитель](./builder.md)
- [Прототип](./prototype.md)
- [Одиночка](./singletone.md)
