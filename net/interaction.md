# Взаимодействие по сети

# DNS

Первый шаг к загрузке страницы - это DNS запрос. Этот запрос кешируется в браузере. Используют [DNS протокол](./models/tcp-ip/1.application/dns.md). Позволяет получить ip по адресу страницы. Запрос будет производиться дял каждого уникального ресурс

# TSP

TSP-рукопожатие, установка соединения, по соединению [http](./models/tcp-ip/1.application/http.md) SYN, SYN-ACK, ACK(Aknowledgements) обмен 3 сообщениям

# TLS

Следующее рукопожатие - [TLS](./models/tcp-ip/2.transport/ssl-tls.md) определяется шифр. После этого шага браузер делает запрос

# ответ на запрос

Первый пакет данных 14кб - проверка скорости соединения, будет увеличивать в 2 раза окно
