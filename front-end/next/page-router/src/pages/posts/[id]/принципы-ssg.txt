------------------------------Static Site Generation (SSG) ------------------------------

getStaticProps:
Для страниц, которые формируются в процессе build'а приложения

Используется:
- Для данных доступных перед запросом
- headless CMS
- Страница должна быть пре-рендерена
- Страница публична кэширована

Запускается:
- при build'е приложения
- на фоне при fallback: true
- перед начальным рендером, если  fallback: blocking
- в фоновом режиме при revalidate
- при вызове revalidate()

!!!Не имеет доступ к query параметрам (использовать middleware)

getStaticPaths - идет в паре с getStaticProps для динамических роутов
Используется:
- данные идут из headless CMS
- данные идут из DB
- данные ищ файловой системы
- Кэшируются публично
- Страница должна быть пре-рендерена

Запускается:
- при build'е приложения
- на фоне при fallback: true
- перед начальным рендером, если  fallback: blocking
