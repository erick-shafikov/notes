# Claude Code — инструкции для проекта notes

## Общение в чате

При объяснении математики в чате использовать ASCII-нотацию вместо LaTeX-символов — их тяжело читать в терминале.

Примеры замен:
- `∇f_i(x)` вместо `$\nabla f_i(x)$`
- `||x - y||` вместо `$\|x - y\|$`
- `x~` или `x_snapshot` вместо `$\tilde{x}$`
- `f*` вместо `$f^*$`
- `E[v]` вместо `$\mathbb{E}[v]$`

**В файлах конспектов (.md) оставлять LaTeX как есть** — там формулы рендерятся корректно.
