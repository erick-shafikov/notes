# Claude Code — инструкции для проекта notes

## Общение в чате

При объяснении математики в чате использовать ASCII-нотацию вместо LaTeX-символов — их тяжело читать в терминале.

Примеры замен:
- `∇f_i(x)` вместо `$\nabla f_i(x)$`
- `||x - y||` вместо `$\|x - y\|$`
- `x~` или `x_snapshot` вместо `$\tilde{x}$`
- `f*` вместо `$f^*$`
- `E[v]` вместо `$\mathbb{E}[v]$`

**В файлах конспектов (.md) оставлять LaTeX как есть** — там формулы рендерятся корректно.

## Нумерация файлов

Файлы внутри тематических папок нумеруются префиксом (`1-`, `2-`, `3-` и т.д.) и отражают логический порядок изучения материала.

При добавлении нового файла (из фото или перенос существующего) нужно определить, куда он логически вписывается:

- **В конец** — если тема идёт после всех существующих (просто берёт следующий номер).
- **В середину** — если тема должна стоять между существующими файлами; тогда нужно сдвинуть нумерацию последующих файлов и обновить все ссылки на переименованные файлы.

Перед добавлением всегда смотреть на содержимое папки и решать, куда файл встаёт по смыслу, а не просто добавлять в конец.
