# структура директорий

- Домашние (home directories): /root, /home, /username
- Пользовательские файлы, которые можно запускать: /bin, /usr/bin, usr,local/bin
- Системные файлы, которые можно запускать: /sbin, /user/sbin, /usr/local/sbin
- системные девайсы: /media, /mnt
- конфигурация: /etc
- временные: /tmp
- ядро /boot
- сервер /var, /srv
- системные /proc, /sys
- библиотечные /lib, /usr/lib, /usr/local/lib

/
├── bin/ → Базовые команды (ls, cp, mv, cat)
├── boot/ → Загрузочные файлы (ядро, GRUB)
├── dev/ → Файлы устройств (диски, терминалы)
├── etc/ → Конфигурационные файлы системы
├── home/ → Домашние папки пользователей
│ ├── erik/
│ └── alex/
├── lib/ → Системные библиотеки
├── lib64/ → 64-битные библиотеки
├── media/ → Точки монтирования внешних устройств (USB, DVD)
├── mnt/ → Временные точки монтирования
├── opt/ → Дополнительное ПО (например, /opt/google/)
├── proc/ → Виртуальная ФС с инфой о процессах и ядре
│ ├── cpuinfo
│ ├── meminfo
│ └── [pid]/ → Процессы по ID
├── root/ → Домашняя папка суперпользователя (root)
├── run/ → Временные PID и сокеты, активные при запуске
├── sbin/ → Системные утилиты для администратора
├── srv/ → Данные сервисов (www, ftp)
├── sys/ → Виртуальная ФС с инфой об устройствах и ядре
├── tmp/ → Временные файлы (очищаются при перезапуске)
├── usr/ → Пользовательские приложения и библиотеки
│ ├── bin/ → Большинство программ
│ ├── sbin/ → Админские утилиты
│ ├── lib/ → Библиотеки
│ ├── share/ → Общие данные (иконки, man-страницы)
│ └── local/ → ПО, установленное вручную (не из пакетов)
└── var/ → Переменные данные
├── log/ → Логи системы и приложений
├── tmp/ → Временные файлы (живут дольше /tmp)
├── cache/ → Кэш программ
├── spool/ → Очереди заданий (печатных, почтовых)
└── lib/ → Данные сервисов (dpkg, mysql)
