# Числовые характеристики непрерывных случайных величин

Пусть $X$ — непрерывная СВ с [плотностью распределения](4-probability-density.md) $\omega(x)$ (= $f(x)$).

## Математическое ожидание

**Первый начальный момент:**

$$\alpha_1 = M(X) = \int_{-\infty}^{+\infty} x\,\omega(x)\,dx$$

**Связь с выборкой.** По $N$ наблюдениям $a_1, a_2, \ldots, a_N$ вычисляют выборочное среднее:

$$m_x = \frac{1}{N}\sum_{i=1}^N a_i$$

При $N \to \infty$ оно сходится к теоретическому:

$$\lim_{N \to \infty} \frac{1}{N}\sum_{i=1}^N a_i = \int_{-\infty}^{+\infty} x\,\omega(x)\,dx = M(X)$$

## Медиана

**Медиана** $M_e$ — значение, делящее распределение пополам:

$$P(X < M_e) = P(X > M_e) = 0{,}5$$

Для нахождения: $F(M_e) = 0{,}5$.

_Пример для дискретной СВ._ Значения $1, 2, 5, 8, 13$ → $M_e = 5$ (средний элемент). При этом $m_x = \frac{1}{5}(1+2+5+8+13) = 5{,}8 \neq M_e$.

## Мода

**Мода** $M_o$ — наиболее вероятное значение СВ, т.е. точка максимума плотности $\omega(x)$.

- Распределение **унимодальное**, если максимум один.
- **Бимодальное** — два локальных максимума.
- **Антимодальное** — минимум внутри (плотность убывает и возрастает).

При симметричном унимодальном распределении: $M = M_e = M_o$.

## Начальные моменты

$$\alpha_k = \int_{-\infty}^{+\infty} x^k\,\omega(x)\,dx \quad \text{— $k$-й начальный момент}$$

В частности: $\alpha_1 = M(X)$, $\alpha_2 = M(X^2)$.

## Центральные моменты. Дисперсия

**Центрированная СВ:** $\hat{X} = X - m_x$, $\;M(\hat{X}) = 0$.

**$s$-й центральный момент:** $\mu_s = M\!\left[(\hat{X})^s\right] = M\!\left[(X - m_x)^s\right]$.

**Дисперсия** — второй центральный момент, характеризует разброс:

$$\sigma^2 = \mu_2 = M\!\left[(X - m_x)^2\right] = \int_{-\infty}^{+\infty}(x - m_x)^2\,\omega(x)\,dx$$

Удобная для вычисления формула (раскрыть скобки):

$$D(X) = \sigma^2 = \int_{-\infty}^{+\infty} x^2\,\omega(x)\,dx - \bigl[M(X)\bigr]^2 = \alpha_2 - \alpha_1^2$$

**Среднеквадратическое отклонение (СКО):**

$$\sigma(X) = \sqrt{D(X)}$$

**Выборочная дисперсия** (при отсутствии теоретического распределения):

$$\hat{\sigma}^2 = \frac{1}{N-1}\sum_{i=1}^N (a_i - m_x)^2$$

Делитель $N-1$ (а не $N$) даёт несмещённую оценку.

## Пример

**Дано:** $F(x) = \begin{cases} 0, & x < 1 \\ 1 - \dfrac{1}{x^3}, & x \geq 1 \end{cases}$. Найти $M(X)$, $D(X)$, $\sigma(X)$.

**Плотность:** $f(x) = F'(x) = \begin{cases} 0, & x < 1 \\ \dfrac{3}{x^4}, & x \geq 1 \end{cases}$

**Математическое ожидание:**

$$M(X) = \int_1^{+\infty} x \cdot \frac{3}{x^4}\,dx = 3\int_1^{+\infty} x^{-3}\,dx = 3\cdot\left[-\frac{1}{2x^2}\right]_1^{+\infty} = 3 \cdot \frac{1}{2} = \frac{3}{2}$$

**Дисперсия:**

$$\int_1^{+\infty} x^2 \cdot \frac{3}{x^4}\,dx = 3\int_1^{+\infty} x^{-2}\,dx = 3\cdot\left[-\frac{1}{x}\right]_1^{+\infty} = 3$$

$$D(X) = 3 - \left(\frac{3}{2}\right)^2 = 3 - \frac{9}{4} = \frac{3}{4}$$

**СКО:**

$$\sigma(X) = \sqrt{\frac{3}{4}} = \frac{\sqrt{3}}{2}$$
