# Непрерывная случайная величина. Функция распределения

**Непрерывная случайная величина** — величина, которая может принимать любые значения из некоторого промежутка (рост, расстояние, время, размеры физических объектов).

## Функция распределения

**Функция распределения** случайной величины $X$ — это функция $F(x)$, определённая для каждого $x \in \mathbb{R}$ как вероятность того, что $X$ примет значение, меньшее $x$:

$$F(x) = P(X < x)$$

Свойства:

1. $0 \leq F(x) \leq 1$ для всех $x$
2. $F(x)$ — непрерывная неубывающая функция
3. $\displaystyle\lim_{x \to -\infty} F(x) = 0$, $\quad\displaystyle\lim_{x \to +\infty} F(x) = 1$
4. Вероятность попадания в промежуток:

$$P(a \leq X \leq b) = F(b) - F(a)$$

_Пример._ Дана функция распределения:

$$F(x) = \begin{cases} 0, & x \leq 0 \\ x^2, & 0 < x \leq 1 \\ 1, & x > 1 \end{cases}$$

Найти: а) $P(X < 0{,}5)$, б) $P(0{,}25 \leq X \leq 0{,}75)$, в) $P(X > 0{,}3)$.

**а)** При $x \leq 0$ имеем $F(x) = 0$, поэтому:

$$P(X < 0{,}5) = F(0{,}5) - F(0) = 0{,}5^2 - 0 = 0{,}25$$

**б)**

$$P(0{,}25 \leq X \leq 0{,}75) = F(0{,}75) - F(0{,}25) = 0{,}75^2 - 0{,}25^2 = 0{,}5625 - 0{,}0625 = 0{,}5$$

**в)** Используем дополнение и тот факт, что $F(0) = 0$:

$$P(X > 0{,}3) = 1 - P(X \leq 0{,}3) = 1 - \bigl(F(0{,}3) - F(0)\bigr) = 1 - 0{,}3^2 = 0{,}91$$
