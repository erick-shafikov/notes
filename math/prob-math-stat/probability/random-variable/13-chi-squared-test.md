# Критерий согласия χ² (Пирсона)

Критерий определяет, **можно ли вообще** описывать данные выбранной кривой распределения. Применяется к гистограмме: левая картина — нормальный закон, правая — равномерный.

## Гипотеза H₀

$H_0$ — гипотеза о том, что кривая $\omega(x)$ **корректно описывает** данные гистограммы.

## Частоты

Выборка объёма $n$ разбита на $k$ интервалов. В $i$-м интервале $m_i$ наблюдений:

- **Эмпирическая** (наблюдаемая) вероятность: $p_i^* = m_i/n$, $\;i = 1, 2, \ldots, k$
- **Теоретическая** (по выбранному закону): $p_i = \omega(x_i)\cdot\Delta x_i$

Условие нормировки: $\displaystyle\sum_{i=1}^k p_i^* = 1$, поэтому $p_k^* = 1 - \displaystyle\sum_{i=1}^{k-1} p_i^*$.

## Статистика χ²

Взвешенная сумма квадратов отклонений:

$$U = \sum_{i=1}^k c_i\,(p_i^* - p_i)^2, \quad c_i = \frac{n}{p_i}$$

$$\chi^2 = U = \sum_{i=1}^k \frac{n(p_i^* - p_i)^2}{p_i} = \sum_{i=1}^k \frac{(m_i - n p_i)^2}{n p_i}$$

При верной $H_0$ и больших $n$ статистика $\chi^2$ имеет **распределение Пирсона** (хи-квадрат).

## Степени свободы

$$r = k - s - 1$$

где $s$ — число параметров, оценённых из данных. Для нормального закона ($m$ и $\sigma^2$): $s = 2$, поэтому:

$$r = k - 3$$

## Принятие решения

1. Задать уровень значимости $\lambda$ — вероятность ошибки первого рода (отвергнуть верную $H_0$).
2. По таблице найти критическое значение $\chi^2_{\text{кр}}(r, \lambda)$.
3. Вычислить наблюдаемое $\chi^2$ по формуле выше.

$$\chi^2 < \chi^2_{\text{кр}} \implies \text{принять } H_0 \qquad \chi^2 \geq \chi^2_{\text{кр}} \implies \text{отвергнуть } H_0$$

Чем меньше $\chi^2$, тем лучше теоретическая кривая описывает данные.
