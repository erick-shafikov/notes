# Совместные события. Теорема сложения

События называются **совместными**, если они могут произойти одновременно, и **несовместными**, если наступление одного исключает наступление другого — их пересечение пусто.

## Вероятность суммы двух событий

Для **несовместных** событий $A$ и $B$ вероятность того, что произойдёт хотя бы одно из них, равна сумме их вероятностей:

$$P(A + B) = P(A) + P(B)$$

Для **совместных** событий это равенство нарушается, потому что пересечение $AB$ учитывается дважды — по одному разу в $P(A)$ и $P(B)$. Его нужно вычесть:

$$P(A + B) = P(A) + P(B) - P(AB)$$

где $P(AB)$ — вероятность одновременного наступления обоих событий. Формула для несовместных событий является частным случаем: при $P(AB) = 0$ она сводится к простой сумме.

*Пример 1 (несовместные).* Бросают игральный кубик. $A$ — выпало чётное число, $B$ — выпало нечётное. События несовместны, так как число не может быть одновременно чётным и нечётным. $P(A) = \frac{3}{6} = \frac{1}{2}$, $P(B) = \frac{1}{2}$.

$$P(A + B) = \frac{1}{2} + \frac{1}{2} = 1$$

Результат ожидаем: одно из двух точно выпадет.

*Пример 2 (совместные).* Из колоды в 52 карты вытягивают одну. $A$ — карта червовая, $B$ — карта фигурная (валет, дама, король). $P(A) = \frac{13}{52}$, $P(B) = \frac{12}{52}$. Пересечение — червовые фигуры (валет, дама, король червей): $P(AB) = \frac{3}{52}$.

$$P(A + B) = \frac{13}{52} + \frac{12}{52} - \frac{3}{52} = \frac{22}{52} = \frac{11}{26} \approx 0{,}42$$

## Вероятность суммы трёх событий

Обобщение на три события получается по тому же принципу включений и исключений: сначала прибавляются вероятности каждого события, затем вычитаются вероятности всех попарных пересечений (они вошли лишний раз), а тройное пересечение возвращается обратно, поскольку было вычтено трижды:

$$P(A + B + C) = P(A) + P(B) + P(C) - P(AB) - P(AC) - P(BC) + P(ABC)$$

Эта структура продолжается для любого числа событий и известна как формула включений и исключений.

*Пример.* Студент решает тест из трёх независимых вопросов. Вероятность верно ответить на каждый: $P(A) = 0{,}8$, $P(B) = 0{,}7$, $P(C) = 0{,}6$. Найти вероятность того, что он ответит хотя бы на один вопрос правильно. Из-за независимости: $P(AB) = 0{,}56$, $P(AC) = 0{,}48$, $P(BC) = 0{,}42$, $P(ABC) = 0{,}336$.

$$P(A+B+C) = 0{,}8 + 0{,}7 + 0{,}6 - 0{,}56 - 0{,}48 - 0{,}42 + 0{,}336 = 0{,}976$$

Тот же результат проще получить через дополнение: $1 - P(\bar{A})P(\bar{B})P(\bar{C}) = 1 - 0{,}2 \cdot 0{,}3 \cdot 0{,}4 = 1 - 0{,}024 = 0{,}976$.
