# Байесовский метод построения оценок

Дано измерение: $y_1, y_2, \ldots, y_n$, где $y_k = \theta + \eta_k$, $\eta_k$ — шум

Вводим априорное распределение параметра: $\theta \sim w(\theta)$

После наблюдения $y$ параметр $\theta$ рассматривается как случайная величина с апостериорным распределением $w(\theta \mid y)$, которое показывает, какие значения $\theta$ более вероятны с учётом данных

Оценка: $\hat{\theta} = \mu(y)$

Ошибка: $\varepsilon = \theta - \hat{\theta}$

Функция потерь (квадратичная): $L(\theta, \hat{\theta}) = (\theta - \hat{\theta})^2$

Байесовский подход: нужно минимизировать среднюю ошибку по распределению $\theta$, а не для фиксированного значения

Байесовский риск: $R(\hat{\theta}) = \int (\theta - \hat{\theta})^2 \cdot w(\theta \mid y)\, d\theta$

Минимизируем риск по $\hat{\theta}$:

$$\frac{d}{d\hat{\theta}} \int (\theta - \hat{\theta})^2 w(\theta \mid y)\, d\theta = 0$$

Переносим производную под знак интеграла:

$$\int 2(\hat{\theta} - \theta) \cdot w(\theta \mid y)\, d\theta = 0$$

$$\int (\hat{\theta} - \theta) \cdot w(\theta \mid y)\, d\theta = 0$$

Раскрываем:

$$\hat{\theta} \int w(\theta \mid y)\, d\theta - \int \theta \cdot w(\theta \mid y)\, d\theta = 0$$

Так как $\int w(\theta \mid y)\, d\theta = 1$, получаем:

$$\hat{\theta} = \int \theta \cdot w(\theta \mid y)\, d\theta$$

Итог: $\hat{\theta}(y) = \mathbb{E}[\theta \mid y]$

Где $\mathbb{E}[\theta \mid y]$ — это математическое ожидание (среднее значение) $\theta$ по апостериорному распределению

Интуиция: после наблюдения $y$ есть распределение вероятных значений $\theta$, и оптимальная оценка при квадратичной ошибке — это центр этого распределения (его среднее значение)

## Нахождение конкретной оценки (МНК как частный случай)

Условие экстремума апостериорного распределения:

$$\sum_{i=0}^{n} (y_i - \theta) = 0$$

Раскрываем сумму:

$$\sum_{i=1}^{n} y_i - \sum_{i=1}^{n} \theta = 0 \quad \Rightarrow \quad n\theta = \sum_{i=1}^{n} y_i$$

Откуда:

$$\boxed{\theta = \frac{1}{n} \sum_{i=1}^{n} y_i}$$

То есть при существовании от апоста до апосте расширение будет давать то же среднее арифметическое, каким бы ни были наблюдения.

## Апостериорное распределение (Байесовское обновление)

По теореме Байеса:

$$w(\theta \mid y) = w(y \mid \theta) \cdot w(\theta)$$

При гауссовом шуме и гауссовом априорном распределении (апостериорное = произведение гауссиан):

$$w(\theta \mid y) = \frac{1}{\sigma_0 (\sigma\sqrt{2\pi})^n} \exp\!\left\{ -\frac{1}{2\sigma^2} \sum_{i=1}^{n}(y_i - \theta)^2 - \frac{1}{2\sigma_0^2}(\theta - \theta_0)^2 \right\}$$

Дифференцируем и приравниваем к нулю — получаем байесовскую оценку:

$$\boxed{\hat{\theta} = \frac{\sigma_\varepsilon^2}{\sigma_\varepsilon^2 + \sigma_0^2}\,\theta_0 + \frac{\sigma_0^2}{\sigma_\varepsilon^2 + \sigma_0^2}\,\hat{\theta}_{\text{мл}}}$$

где $\sigma_\varepsilon^2 = \dfrac{\sigma^2}{n}$ — дисперсия ошибки оценки (дисперсия выборочного среднего).

Интерпретация: байесовская оценка — это **взвешенная сумма** априорного среднего $\theta_0$ и оценки максимального правдоподобия $\hat{\theta}_{\text{мл}}$. Веса определяются соотношением дисперсий.

## Априорное распределение

Если есть дополнительные сведения о $\theta$, используется гауссово априорное распределение:

$$w(\theta) = \frac{1}{\sigma_0\sqrt{2\pi}}\, e^{-\dfrac{(\theta - \theta_0)^2}{2\sigma_0^2}}$$

где $\theta_0$ — априорное среднее, $\sigma_0^2$ — априорная дисперсия (степень неопределённости о $\theta$ до наблюдений).
